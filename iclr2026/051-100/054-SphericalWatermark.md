# Spherical Watermark: Encryption-Free, Lossless Watermarking for Diffusion Models

> **Venue**: ICLR 2026 Oral | **ArXiv**: N/A | **SphericalWatermark**

扩散模型（Diffusion Models）的飞速发展彻底改变了图像生成领域，以 Stable Diffusion 为代表的文生图系统已经能够合成高度逼真的图像，并在创意产业中得到广泛应用。然而，这类技术的普及也带来了严峻的内容安全挑战：恶意用户可以利用扩散模型批量生成虚假图像、传播虚假信息，进而侵蚀公众信任、引发法律和伦理纠纷。为了追踪 AI 生成内容的来源，数字水印技术被认为是一种可行的解决思路。传统的图像水印方法，无论是在空域（如 LSB 嵌入）还是频域（如 DWT-DCT 方案）对图像进行操作，都会不可避免地改变图像的统计分布，引入可感知的伪影，从而降低视觉质量。另一类方法通过微调扩散模型本身（如 Fine-tuning Stable Diffusion 的解码器）来植入水印，虽然无需修改推理流程，但代价高昂且缺乏灵活性。还有基于频域隐写的方案（如 Tree-Ring）以及通过 fine-tuning 引入触发机制的方案（如 SleeperMark），虽能在一定程度上保留语义一致性，但均存在可感知的图像质量损失。

近年来，"无损水印"或称"不可检测水印"的概念受到关注。这类方法的核心思想是在扩散模型的潜在高斯噪声输入端植入水印，从而在不修改预训练模型权重的前提下实现水印嵌入，理论上不引入任何视觉质量损失。Gaussian Shading（Yang et al., 2024）是这一方向的代表性工作，它利用重复编码与流密码将水印映射为标准高斯噪声；然而其每张图像需要存储一个独立的密钥和随机数（nonce），在大规模 API 部署场景下带来了极高的密钥管理开销。随后，PRC Watermark（Gunn et al., 2025）以固定密钥的伪随机纠错码（Pseudorandom Error-Correcting Codes, PRC）取代流密码，在一定程度上缓解了存储问题，但其置信传播解码（belief-propagation decoding）引入了巨大的计算延迟，且对码率与纠错能力的权衡需要精细调参，在面对强攻击时会遭遇不可约的误差下界而彻底失效。本文所提出的 Spherical Watermark 正是为了解决上述所有痛点而设计：无需加密、无需每图存储密钥、无损嵌入、高效提取、对常见攻击鲁棒。

Spherical Watermark 的核心框架由三个可逆模块串联构成：二进制嵌入模块（Binary Embedding Module $\mathcal{B}$）、球面映射模块（Spherical Mapping Module $\mathcal{S}$）以及扩散集成模块（Diffusion Integration Module $\mathcal{G}$）。整个系统分为离线构建阶段和在线运行阶段。在离线构建阶段，模型开发者生成一个固定的"签名"$\mathcal{K} = (\mathbf{T}, \mathbf{C})$，其中 $\mathbf{T}$ 为嵌入矩阵（Embedding Matrix），$\mathbf{C}$ 为旋转矩阵（Rotation Matrix）。在在线运行阶段，每次 API 调用时，该签名被自动应用，将用户相关的水印嵌入扩散模型的潜在噪声输入中，确保生成图像携带可追踪的来源标记。

具体地，水印消息 $\mathbf{m} \in \{0,1\}^{l_m}$（编码了用户 ID、时间戳等 API 元数据）经过以下流程转化为标准高斯噪声：首先，水印预处理阶段将 $\mathbf{m}$ 重复 $N$ 次并拼接随机填充向量 $\mathbf{r} \in \{0,1\}^{l_r}$，得到高熵序列 $\mathbf{x} = [\mathbf{m}, \mathbf{m}, \cdots, \mathbf{m}, \mathbf{r}]^\top \in \{0,1\}^{l_x}$，其中 $l_x = N \times l_m + l_r$。接着，二进制嵌入模块在二进制域 $\mathbb{F}_2$ 上执行矩阵乘法 $\mathbf{z}^{(1)} = \mathbf{T}\mathbf{x}$，其中嵌入矩阵 $\mathbf{T}$ 的设计确保每个水印位都与随机填充混合，从而引入额外随机性、降低重复结构带来的统计相关性。随后，球面映射模块将二进制向量转换为高斯噪声，具体分三步：将 $\mathbf{z}^{(1)}$ 映射到连续域得到 $\mathbf{v} = 2\mathbf{z}^{(1)} - \mathbf{1}$，再归一化到单位球面 $\mathbf{z}^{(2)} = \mathbf{v} / \|\mathbf{v}\|_2$，然后施加正交旋转 $\mathbf{z}^{(3)} = \mathbf{C}\mathbf{z}^{(2)}$，最后对 $\chi^2(l_x)$ 分布随机采样半径 $r$（满足 $r^2 \sim \chi^2(l_x)$），将球面上的单位向量缩放为 $\mathbf{z}_w = r \cdot \mathbf{z}^{(3)}$，得到最终的含水印高斯噪声。扩散集成模块将 $\mathbf{z}_w$ 作为初始噪声输入 Stable Diffusion，通过求解概率流 ODE（Probability-Flow ODE）$dz_t/dt = f_t(z_t) - \frac{1}{2}g_t^2 \nabla_{z_t} \log p_t(z_t)$ 进行去噪，最终经 VAE 解码器生成含水印图像 $O_w$。水印提取时，反向流程利用 DDIM 反演恢复初始噪声估计，再经逆旋转、反二进制映射及多数投票译码（Majority Vote Decoding）还原水印消息。

在理论分析方面，论文严格证明了球面映射后的水印噪声 $\mathbf{z}_w$ 在统计上与标准多元高斯分布 $\mathcal{N}(\mathbf{0}, \mathbf{I}_{l_x})$ 不可区分。关键的理论结果如下：定理 3.1 证明，在 $\mathbf{m}$ 和 $\mathbf{r}$ 均为独立 Bernoulli$(1/2)$ 比特的条件下，$\mathbf{z}^{(1)}$ 中每个分量均服从 Bernoulli$(1/2)$ 分布，且 $\mathbf{z}^{(1)}$ 具有 2-wise 和 3-wise 独立性。定理 3.2 进一步证明，归一化后的 $\mathbf{z}^{(2)}$ 满足球面 3-设计（Spherical 3-Design）的条件——球面 $t$-设计是单位球面上的有限点集，对所有次数不超过 $t$ 的实多项式，其有限均值等于连续球面均匀分布的积分均值，因此 $\mathbf{z}^{(2)}$ 可视为球面均匀分布的近似。引理 3.3 指出，对球面 3-设计施加正交旋转后，$\mathbf{z}^{(3)}$ 仍是球面 3-设计，其各分量均值为零、方差为 $1/l_x$，且随 $l_x \to \infty$，各分量的边际分布收敛到 $\mathcal{N}(0, 1/l_x)$。引理 3.4 则利用多元正态分布的极坐标分解定理——即 $\mathbf{n} \sim \mathcal{N}(\mathbf{0}, \mathbf{I}_n)$ 当且仅当 $\mathbf{n} = r \cdot \mathbf{u}$，其中 $r^2 \sim \chi^2(n)$，$\mathbf{u}$ 均匀分布在 $S^{n-1}$ 上且与 $r$ 独立——严格证明了 $\mathbf{z}_w = r \cdot \mathbf{z}^{(3)}$ 服从标准多元正态分布。这一系列证明构成了水印噪声分布与标准高斯分布精确到三阶矩的匹配性保证，从而在理论层面确立了方法的无损性。

实验设计方面，本文以 Stable Diffusion v1.5 和 v2.1 为骨干模型，生成分辨率为 $512 \times 512$ 的彩色图像，潜在空间维度为 $4 \times 64 \times 64$（即 $l_x = 16384$）。图像生成采用 50 步 DPM-Solver++ 推理，水印提取采用 50 步 DDIM 反演。默认超参数设置为 $N=31$（重复次数）、$l_m = 512$ 比特（水印长度）、$l_r = 512$（填充长度）、$s=1$（行稀疏度）。实验在四块 NVIDIA RTX 4090 GPU 上完成。基线方法覆盖传统水印（DwtDct、DwtDctSvd、RivaGAN，均嵌入 32 比特）以及基于潜在空间的水印（Tree-Ring、Gaussian Shading、PRC Watermark，均评估 512 比特容量）。评估数据集为从 MS-COCO val2017 随机采样的 1000 条文本提示（COCO）和来自 Stable Diffusion Prompts 数据集的 1000 条提示（SDP）。核心评估指标包括：不可检测性（FID 分数、针对有水印/无水印样本的分类器准确率）以及追踪准确率（比特级准确率 ACC 和 TPR@1%FPR，后者简写为 TPR）。

实验结果令人印象深刻。在不可检测性方面，FID 实验（表1）表明，仅 PRC Watermark 和 Spherical Watermark 的 FID 分数与原始无水印生成相当（约 46–50），而 DwtDct、DwtDctSvd、RivaGAN 以及固定密钥设置下的 Gaussian Shading 均引入了统计分布偏移导致 FID 升高。分类器检测实验进一步验证了这一点：使用双层 MLP 进行潜在空间级分类时，Tree-Ring 的检测准确率高达 100%，Gaussian Shading（固定密钥）为 97%，而 PRC Watermark 和 Spherical Watermark 均接近随机水平（50%），表明潜在空间中的统计分布完全未被破坏；在图像级 ResNet-18 分类器上，两者的检测准确率同样约为 50%。在追踪准确率方面（表2），干净条件（PNG 存储）下，Spherical Watermark 的 ACC 为 99.99%、TPR 为 100%；经过常见后处理攻击（包括高斯噪声、亮度调整、高斯模糊、中值滤波、JPEG 压缩、缩放等）后，ACC 保持在约 95.02%、TPR 约 97.50%，均显著优于 PRC Watermark（分别约 93.52% 和 87.03%）；在 WEvade 对抗攻击下，Spherical Watermark 的 ACC 为 98.12%、TPR 高达 99.83%，比 PRC Watermark（ACC 97.69%、TPR 95.38%）高出显著差距，更比所有有损水印方法高出 10% 以上——这与理论分析吻合：有损水印因引入统计分布偏移而使攻击者可以训练分类器并以此为媒介发起对抗攻击，而无损水印从根本上消除了这一可攻击面。在计算效率方面，Spherical Watermark 的嵌入和提取时间极短，其中提取速度约比 PRC Watermark 快四个数量级（约 $10^4$ 倍），充分体现了无加密设计的工程价值。大容量水印实验（图6a）还表明，在 JPEG-70 压缩条件下，PRC Watermark 的 ACC 随水印容量 $l_m$ 增大而急剧下降，超过 2000 比特时完全失效；而 Spherical Watermark 在全容量范围内均保持稳定的高 ACC。

消融实验从多个维度深入剖析了各模块和超参数的作用。模块消融表明，若去掉二进制嵌入模块 $\mathcal{B}$（仅保留球面映射），潜在噪声在潜在空间分类器上变得极易检测（图6b）；若去掉球面映射模块 $\mathcal{S}$（以 Gaussian Shading 变换替代），在亮度攻击下的鲁棒性急剧下降（图6c）。这说明二进制嵌入负责保证独立性和不可检测性，而球面映射则是实现对非线性变换鲁棒性的关键。超参数消融（图6d）表明，在较宽的参数范围内（$N$、$l_m$、$l_r$、$s$ 各自变化），潜在空间分类器准确率始终维持在约 50%，不可检测性不受影响。行稀疏度 $s$ 增大或重复次数 $N$ 减小会使 ACC 有所下降（表3），但即便在 $s=4$、极端 JPEG-70 压缩条件下，TPR 仍超过 92%。ODE 求解器消融（表4）表明，无论采用 DDIM、PNDM 还是 DPM-Solver++，提取精度基本相同，说明球面映射的内在冗余有效吸收了求解器差异引入的数值误差。时间步消融（表5）进一步证实，生成和反演的时间步数（10 至 50 步）对提取准确率的影响极小。

综合来看，Spherical Watermark 对 AI 生成内容溯源领域具有重要意义。首先，它在理论层面提供了比之前工作更严格的高斯分布近似保证——通过球面 3-设计和 $\chi^2$ 半径缩放实现了精确到三阶矩的分布匹配，而非依赖启发式变换。其次，无加密设计彻底消除了每图密钥存储的工程负担，使水印系统在大规模 API 服务中具有真正的可部署性——一套固定签名 $(\mathbf{T}, \mathbf{C})$ 即可服务所有用户请求，仅依靠用户 ID 编码在水印消息 $\mathbf{m}$ 中实现差异化追踪。第三，方法的无损性从根本上杜绝了基于分布偏移的对抗攻击，这一洞见对水印安全研究具有范式意义：有损水印的质量损失不仅仅是"副作用"，更是其安全性的根本漏洞。第四，该方法具有良好的通用性，理论上可迁移到任何具有高斯先验和可逆映射的生成模型（如 Flow Matching 模型）。本文在 ICLR 2026 上获得口头报告（Oral）认可，反映了学界对这一工作在理论严谨性与实际可用性兼备方面的高度认同，也为 EU AI Act 等监管框架下的 AI 内容溯源基础设施建设提供了坚实的技术基础。
