# MTVCraft: Tokenizing 4D Motion for Arbitrary Character Animation

> **Venue**: ICLR 2026 | **ArXiv**: 2505.10238 | **MTVCraft**

人物动画（Character Animation）是视频生成领域中极具挑战性的任务，其核心目标是依据给定的参考图像与驱动运动序列，生成外观一致、动作真实的人物视频。现有方法的主流范式依赖于将三维骨骼姿态渲染为二维姿势图像（例如 OpenPose 骨架图或 DWPose 关键点热图），再将其作为控制信号注入视频扩散模型。这种基于二维姿态图像的引导方式虽然直观，但存在几个根本性缺陷：首先，姿态图像的渲染需要严格的像素级对齐，即驱动视频中的人物体型、比例与朝向必须与参考图像高度一致，否则会产生明显的形变或遮挡问题；其次，二维渲染丢弃了原始三维运动序列中的深度信息，使得模型在处理复杂空间运动（如前后移动、旋转）时难以获得充分的空间线索；再者，由于二维渲染的感受野局限于像素空间，模型往往是在"逐像素复制"固定形状的姿态图，缺乏对运动语义的真正理解，导致在遇到未见风格（如卡通角色、非人类对象）时泛化能力严重不足。正是针对上述局限，MTVCraft 提出了一种全新的技术路线：直接对原始三维运动序列进行建模，通过将四维运动信息（三维空间坐标加时间维度）量化为紧凑的离散令牌，来替代传统的二维姿态图像作为运动引导信号。

MTVCraft 的核心技术贡献由两大组件构成。第一个组件是四维运动分词器（4D Motion Tokenizer，简称 4DMoT）。4DMoT 采用向量量化变分自编码器（VQVAE）的结构，以 SMPL 参数化骨骼的关节三维坐标序列作为输入，通过编码-量化-解码三阶段将其压缩为离散运动令牌。具体而言，编码器由三个带有时间与关节维度下采样的 ResNet 模块组成，将输入序列映射至 3072 维的潜在编码空间；向量量化层维护一个包含 8192 个码字的码本，采用指数移动平均（EMA，衰减系数 $\lambda = 0.99$）更新策略以稳定训练；解码器则采用与编码器对称的结构将量化编码还原为运动序列。在数据预处理环节，4DMoT 将每帧关节坐标减去第一帧的绝对位置，转化为相对运动表示，使模型聚焦于运动动态本身而非绝对位置，这一设计显著提升了对不同场景尺度的鲁棒性。训练目标由重建损失与承诺损失共同构成：

$$\mathcal{L}_{vq} = \|M - \hat{M}\|_1 + \beta \|E - \text{sg}[C]\|_2^2$$

其中 $M$ 与 $\hat{M}$ 分别为原始与重建运动序列，$E$ 为编码器输出，$C$ 为最近码字，$\text{sg}[\cdot]$ 表示停止梯度操作，$\beta$ 为承诺损失系数。

第二个组件是运动感知视频扩散变换器（Motion-aware Video DiT，简称 MV-DiT）。MV-DiT 以成熟的视频扩散变换器（Video DiT）为骨干，通过三项关键设计将运动令牌融入生成过程。其一，身份保持采用"重复拼接"设计，将参考图像与带噪视频潜变量直接在时间维度拼接，使二者之间能够直接建立时空交互，而无需引入额外的参考分支网络，既减少了参数量又避免了双流架构的对齐负担。其二，四维位置编码将标准的三维旋转位置编码（3D RoPE）扩展至四维空间：对于视觉令牌，编码维度为 $(t, h, w)$；对于运动令牌，编码维度为 $(t, x, y, z)$，其中 $x, y, z$ 对应关节的三维坐标，采用数据集中各关节位置的统计均值作为稳定参考，使四维位置编码在训练与推理之间保持一致的语义对应关系。其三，四维运动注意力以视觉令牌为查询（Query）、以运动令牌为键值对（Key/Value），通过交叉注意力机制实现运动对视觉生成的精确调控，注意力计算过程中对两种模态分别施加各自对应维度的 RoPE，确保位置感知的准确性。此外，MV-DiT 还引入了运动感知无分类器引导（Motion-aware CFG），通过在训练时随机将运动条件替换为可学习的无条件运动令牌 $c_{mo\emptyset}$，使模型在推理时能够通过调节运动引导强度灵活控制生成效果。

在模型规模方面，论文提供了两个版本：MTVCraft-6B 以 CogVideoX-5B 为骨干，移除文本处理分支，每两个 DiT 块插入一个运动注意力模块；MTVCraft-18B 以 Wan-2.1-14B 为骨干，保留文本分支以支持文本与运动的联合控制，通过零填充将运动令牌的 3072 维对齐至骨干网络的 5120 维。训练数据方面，研究团队从公开数据集与网络爬取中收集了 20 万段初始视频片段，经过镜头切割、单人筛选、姿态估计、不确定性过滤、美学评分与光流质量评估等多级严格筛选，最终保留约 3 万段高质量片段（平均每段 600 帧）。MV-DiT 训练在 12 块 H100 GPU 上进行约 3 万次迭代，采用 bfloat16 精度与 DeepSpeed ZeRO-2 策略，总计约 12 H100·天。

实验部分在 TikTok 与 Fashion 两个标准人物动画基准上进行了全面评测，涵盖图像质量（PSNR、SSIM、LPIPS）与视频质量（FID、FVD、FID-VID）共五项指标，推理采用 DDIM 50 步采样，运动引导强度 3.0，文本引导强度 6.0。定量结果表明，MTVCraft-18B 在 TikTok 基准上的 PSNR 为 19.84，SSIM 为 0.779，LPIPS 为 0.217，FID 为 20.70，FVD 为 276.65，全面超越当前最强基线 Unianimate-DiT（PSNR 19.35 / SSIM 0.765 / LPIPS 0.235 / FID 28.47 / FVD 402.14）。消融实验从多个角度验证了设计选择的有效性：以离散量化替换连续编码后 FVD 从 317.21 恶化至 332.97，说明量化对运动表示的规整化作用不可或缺；去除差分运动表示后性能同样下降，验证了相对运动建模的重要性；在位置编码方面，去除所有位置编码使 FVD 暴涨至 548.31，仅使用一维时间 RoPE 或二维空间 RoPE 均不及完整的四维方案，而采用数据集均值关节位置的稳定策略优于动态位置或可学习位置，进一步证明了四维 RoPE 设计的精心权衡。在零样本泛化实验中，尽管模型完全在人类数据上训练，却能够在未见的卡通人物、动物乃至无生命物体上产生高质量的动画效果，展现出超越像素级对齐方法的本质优势——模型真正学习到了运动的语义表示，而非单纯复制姿态图像的像素分布。定性对比也表明，MTVCraft 在处理姿态未对齐（如体型差异较大的参考图像）情况时，能够避免传统二维方法产生的明显形变与伪影。

MTVCraft 的研究意义在于为人物动画乃至更广泛的可控视频生成领域提供了一种新颖的范式：将运动引导从像素空间迁移至语义空间，通过离散令牌的形式使三维运动先验与视频扩散模型实现高效融合。这一思路与语言模型中的离散 token 化理念高度契合，不仅赋予模型更强的泛化能力，也为多模态控制（文本、运动的联合引导）奠定了基础。未来的研究方向包括：扩充训练数据中的手部动作样本以改善当前手部细节欠佳的局限；探索将 4DMoT 扩展至全身多人场景；以及将该框架与更大规模基础视频模型结合，进一步挖掘四维运动令牌在开放世界动画生成中的潜力。
