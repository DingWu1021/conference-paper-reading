# InfoTok: Adaptive Discrete Video Tokenizer via Information-Theoretic Compression

> **Venue**: ICLR 2026 | **ArXiv**: 2512.16975 | **InfoTok**

在视频生成与理解领域，如何高效、准确地对长视频序列进行离散化表示，是整个研究方向的核心基础问题。现有的视频分词器（video tokenizer）普遍采用固定压缩率的策略，将输入视频统一划分为等长的离散 token 序列。这种"一刀切"的方法存在根本性的缺陷：对于信息密度较低的场景（如静态背景、简单纹理），固定压缩率会造成大量冗余 token 的产生；而对于信息复杂度较高的场景（如快速运动、细节丰富的画面），固定压缩率又可能导致信息丢失。随着视频生成模型对超长序列的需求日益增加，token 数量的冗余问题已成为制约模型效率和质量的关键瓶颈。InfoTok 正是在这一背景下被提出，试图从信息论的根本原理出发，为自适应视频分词建立严格的理论框架，并设计出切实可行的工程方案。

该工作的理论出发点来自香农信息论中的信源编码定理。论文作者团队（包括来自 NVIDIA、斯坦福大学、新加坡国立大学等机构的研究者）首先严格证明了现有固定长度分词器的次优性：基于**定理 2.1**（香农信源编码），固定压缩率的分词器相比理论最优的自适应方法，需要显著更多的 token 才能达到等效的重建质量。进一步地，**定理 2.2** 指出，当前常见的数据无关路由策略（如均匀路由）在理论上存在系统性偏差，其所需的 token 序列长度相比最优解可能趋于无穷大。这两个定理从理论层面奠定了自适应分词方案的必要性与优越性。

在方法论层面，InfoTok 的核心创新在于提出了基于证据下界（Evidence Lower Bound，ELBO）的自适应路由机制。具体而言，对于输入视频 $x$，InfoTok 使用 ELBO 近似其负对数似然，从而将每帧视频的信息复杂度量化为一个连续的标量值。token 序列长度 $N_x$ 的分配公式为：

$$N_x = \beta \cdot \frac{\text{ELBO}(x)}{\mathbb{E}[\text{ELBO}(x)]}$$

其中 $\beta$ 是控制平均压缩率的超参数。这一设计的精妙之处在于，ELBO 本身可通过一次前向传播高效计算，无需昂贵的二分搜索或多次解码器调用，将推理开销从原本的 11 次函数评估压缩至 1 次。**定理 3.1** 进一步证明，基于 ELBO 的路由策略在可证明的误差范围内逼近理论最优解，为整个框架提供了严密的理论背书。

在架构设计上，InfoTok 构建于 Cosmos 离散视频分词器之上，在其基础上引入了轻量级的自适应压缩模块（Adaptive Compressor）。该模块由一个基于 Transformer 的编码器-解码器结构组成：编码器首先将输入视频帧转换为固定长度的嵌入向量序列，随后 ELBO 路由器计算每帧的信息复杂度并确定目标 token 数量，最后自适应压缩器根据似然分数保留信息量最高的 token，丢弃冗余部分。对应的解压缩器（Decompressor）则负责在重建阶段将变长 token 序列恢复为固定长度的特征图，使得下游任务无需感知变长输入的存在。整个自适应模块仅增加约 14.6% 的参数量，即可将原有固定长度分词器无缝升级为自适应版本，具有极强的通用性与兼容性。此外，论文还提出了 InfoTok-Flex 变体，通过一个模型同时支持多个压缩率，进一步提升了系统的灵活性。

在 token 选择策略上，InfoTok 采用基于似然的 token 保留机制：对于每一帧，优先保留具有最高 ELBO 值的 token，因为这些 token 对应的区域携带了最丰富的信息。论文通过消融实验对比了多种替代方案，包括从右到左掩码（R2L masking）和空间分散选择（spatial dispersal），结果表明基于似然的选择策略在重建质量上显著优于其他方案，验证了信息论驱动的 token 选择的合理性。

在实验验证部分，作者在主流视频重建基准上进行了全面评估。以 BPP₁₆ = 0.81 的压缩率为基准，InfoTok 相比固定长度基线节省了 20% 的 token 数量，同时保持了等效的重建质量，充分体现了自适应分词的效率优势。在与先前自适应方案 ElasticTok 的对比中，InfoTok 实现了 2.3 倍更优的压缩率，同时在多项关键指标上全面领先：峰值信噪比（PSNR）从 28.26 提升至 30.08，Fréchet 视频距离（FVD）从 141 大幅降低至 49，感知相似度指标 LPIPS 从 0.244 降低至 0.145。这些结果证明，InfoTok 不仅在压缩效率上超越了启发式自适应方法，在重建质量的绝对水平上也取得了显著提升。消融研究进一步确认了 ELBO 路由器的设计合理性，其性能可以匹敌耗时的 oracle 搜索策略，同时将推理复杂度降低了一个数量级。

从更广泛的视角来看，InfoTok 的研究意义超越了视频分词这一具体任务。该工作率先将香农信息论的严格数学框架引入多模态内容的离散表示学习，建立了信息密度与 token 分配之间的定量对应关系，为整个领域提供了新的理论工具。作者明确指出，这一框架的适用范围可以自然延伸至音频、3D 点云等任何存在可变信息密度的模态。在实践层面，InfoTok 能够以近乎零侵入的方式升级现有分词器，使其获得自适应压缩能力，这一特性使其具备了广泛的工程落地价值。在大规模视频生成模型日益普及的当下，token 效率的提升将直接转化为训练成本的降低和推理速度的加快，InfoTok 所提供的理论指导与工程方案对于构建下一代高效视频生成系统具有重要的参考价值。唯一值得关注的局限性在于，ELBO 估计本身需要一次额外的解码器前向传播，以及用于存储二值掩码的约 5% 额外开销，但这些代价相对于所带来的压缩收益而言完全在可接受范围之内。
