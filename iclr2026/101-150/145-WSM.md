# WSM: Decay-Free Learning Rate Schedule via Checkpoint Merging for LLM Pre-training

> **Venue**: ICLR 2026 | **ArXiv**: 2507.17634 | **WSM**

大型语言模型的预训练过程中，学习率调度策略一直是影响最终模型质量的关键因素之一。传统上，余弦衰减（cosine decay）调度方案因其平滑的衰减特性而被广泛采用，但其致命缺陷在于需要预先确定训练总步数，一旦训练中途希望追加新数据，整个衰减曲线便需要重新设计，甚至需要从头训练。为了缓解这一灵活性问题，Warmup-Stable-Decay（WSD）策略应运而生，它在预热阶段与衰减阶段之间引入了一段恒定学习率的稳定训练阶段，使得训练者可以在任意时刻触发衰减。然而WSD方案依然要求人工决定衰减开始的时机、衰减函数的具体形式以及分配给衰减阶段的token数量，这些超参数的选取往往依赖于大量试验成本高昂的消融实验。本文提出的WSM（Warmup-Stable and Merge）框架在此背景下应运而生，其核心思想是彻底消除训练过程中的显式学习率衰减阶段，转而通过检查点融合（checkpoint merging）来实现等效的优化效果，从而在理论严格性与实践灵活性之间寻求新的平衡点。

WSM的方法论核心在于建立学习率衰减与检查点平均之间的形式化数学联系。给定训练过程中从第 $n$ 步到第 $n+k$ 步保存的一系列检查点参数 $\theta_n, \theta_{n+1}, \ldots, \theta_{n+k}$，合并后的检查点可以表示为加权平均：

$$\hat{\theta}_{n+k} = \sum_{j=0}^{k} c_j \theta_{n+j}$$

其中权重满足 $\sum_{j=0}^{k} c_j = 1$。通过将每个中间检查点展开为初始点与梯度更新的叠加形式，可以进一步推导出合并操作等价于对累积梯度施加一组加权系数：

$$\hat{\theta}_{n+k} = \theta_n - \sum_{i=1}^{k} w_i \cdot g_{n+i-1}$$

这一推导揭示了一个深刻的等价关系：检查点合并实质上是对不同时刻梯度的重新加权，而学习率衰减同样是通过调整梯度的有效学习率来实现对不同时刻梯度贡献的重新分配。论文中的定理3.1给出了从目标衰减系数 $\{w_i\}$ 反推检查点权重 $\{c_j\}$ 的解析公式：对末端权重有 $c_k = w_k$，对中间节点有 $c_j = w_j - w_{j+1}$（$j \in [1, k-1]$），而对初始检查点有 $c_0 = 1 - w_1$。这套转换公式与优化器的具体形式无关，无论是SGD、Adam还是其他任何优化方法均适用，赋予了WSM框架极强的通用性。

在具体的算法流程上，WSM分为两个阶段。第一阶段为标准的线性预热阶段，学习率从零线性增长至峰值。第二阶段进入稳定训练与异步合并阶段：在整个训练过程中保持峰值学习率不变，以固定间隔保存检查点，并通过异步方式对最近的 $n$ 个检查点按照理论推导的权重进行合并，得到的合并模型作为评估或下游使用的最终输出。可选地，在某个切换步骤 $T_{\text{switch}}$ 之后，可以切换到高质量退火数据继续训练，进一步提升模型质量。整个训练管道无需任何衰减阶段，实现了真正意义上的"无衰减"持续预训练。

实验部分基于Ling-mini模型进行验证，该模型为一个混合专家（MoE）架构的语言模型，总参数量达16.3B，激活参数量为1.4B，包含20层Transformer、2048维隐藏层表示，每个MoE层包含256个专家，每次激活其中8个，并采用分组查询注意力（GQA）和旋转位置嵌入（RoPE）。训练配置使用AdamW优化器，参数 $\beta_1=0.9$，$\beta_2=0.95$，权重衰减为0.1，峰值学习率为 $4.78\times10^{-4}$，批次大小为2048（每批约1600万个token）。在已经完成10.2万亿token持续训练的基础模型之上，再分支训练4000亿token进行比较实验。评估基准覆盖通用知识（ARC、AGIEval、BBH等）、语言理解（RACE、SQuAD、TriviaQA）、专业知识（MMLU、MMLU-Pro、C-Eval）、数学推理（GSM8K、MATH、GAOKAO）以及代码生成（HumanEval、MBPP、LiveCodeBench）等多个维度。

主要实验结果表明，WSM在所有基准测试上均显著优于WSD基线，在MATH数学推理任务上提升+3.5%，在HumanEval代码生成任务上提升+2.9%，在MMLU-Pro专业知识任务上提升+5.5%，基础模型平均提升+2.04%，指令微调模型平均提升+1.86%。值得注意的是，这种性能优势在经过有监督微调（SFT）之后依然得以保留，说明WSM带来的提升并非来自对特定评估基准的过拟合，而是反映了模型泛化能力的真实增强。

消融实验方面，论文最重要的发现是合并窗口长度（merge duration）——即参与合并的检查点所覆盖的训练步数窗口——是影响合并效果的最关键因素，其重要性远超检查点保存间隔和合并检查点的数量。更长的合并窗口通常带来更好的性能，但边际收益递减，与传统衰减阶段的行为高度一致。在合并算法的选择上，实验验证了理论预期的性能层次：近似逆平方根衰减的"1-sqrt merge"方案表现最优，近似线性衰减的均值平均方案次之，近似指数衰减的EMA方案则明显落后。这一结果印证了理论框架的有效性——在WSD设置中表现优秀的衰减策略，其对应的检查点合并方案同样更为有效。此外，实验还验证了合并与衰减之间的不可叠加性：无论先衰减后合并还是先合并后衰减，均无法取得协同增益，说明二者是达成相似优化目标的两条独立路径，而非互补手段。

论文还通过t-SNE降维可视化对参数动态进行了深入分析，发现在学习率衰减阶段，模型参数逐渐向合并模型所处的参数空间收敛，而该参数区域对应更优的下游性能。作者将此类比为：恒定学习率训练如同在峡谷中来回震荡前行，而检查点合并则如同找到峡谷底部流淌的河流，沿河流方向高效收敛。从MoE负载均衡的角度看，WSM合并后的检查点相较于WSD衰减后的检查点展现出更好的专家路由均衡性，训练损失略高但下游性能更优，作者将此解读为泛化能力增强的体现。

在实际应用场景中，WSM支持离线合并和在线合并两种模式。离线合并保留完整的历史检查点，方便研究者在不影响主训练进程的前提下系统探索不同合并策略，等价于以较低的存储开销换取对超参数空间的充分探索；在线合并则采用滑动窗口方式，在确定最优策略后可用于存储受限的生产环境。整体而言，检查点历史记录的存储开销相较于通过多次完整训练确定最优衰减策略所需的计算预算而言微不足道。此外，WSM还具备一个颇具价值的工程特性：在训练的不同阶段，WSM合并模型的性能与真实学习率衰减后的性能高度吻合，使其可作为预训练过程中模型潜力的实时评估工具，避免了大量专门针对评估而发起的昂贵衰减实验。

综合来看，WSM框架在理论贡献、实证洞见与实践价值三个维度上均有所建树。理论层面，它在学习率衰减与检查点融合之间建立了严格的数学对应关系，为这两个此前被视为独立的训练技术提供了统一的理解框架；实证层面，它发现合并窗口长度是主导性超参数，并验证了合并算法层次与衰减策略层次的一致性；实践层面，它通过消除衰减阶段将预训练过程简化为一个更加自主、持续的流程，在多个重要基准上取得了超越WSD的显著性能提升。WSM代表了一种新颖而有前景的大语言模型预训练学习率调度思路，有望成为未来超大规模模型持续预训练的重要基础设施之一。
