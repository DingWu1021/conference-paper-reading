# PixNerd: Pixel Neural Field Diffusion

> **Venue**: ICLR 2026 | **ArXiv**: 2507.23268 | **PixNerd**

现有主流扩散模型（Diffusion Model）普遍依赖于预训练变分自编码器（VAE）所压缩的潜在空间，将图像先编码为低分辨率的隐变量再在其上进行扩散建模。这一两阶段范式虽然在计算上高效，却在实践中不可避免地引入了累积误差与解码伪影：VAE 的重建质量对最终生成图像的上限构成硬约束，而 VAE 训练本身也带来了相当的工程复杂度。另一条路线则尝试回归像素空间直接建模，但由于每个 patch 被展平成一维 token，原始像素分辨率所对应的序列长度极大地增加，自注意力的计算开销随序列长度平方增长，导致现有像素扩散模型不得不依赖复杂的级联流水线（cascade pipeline）来逐步提升分辨率，整体架构臃肿、推理延迟高昂。PixNerd 正是在这一背景下应运而生，其核心动机是：能否在像素空间中做到单阶段、端到端的高质量图像生成，同时保持与潜在扩散模型可比的效率？

PixNerd 提出了一种像素神经场扩散（Pixel Neural Field Diffusion）框架，将大 patch 扩散变换器（large-patch diffusion transformer）与隐式神经场（implicit neural field）解码器有机结合，实现了无需 VAE、无需级联流水线的单阶段像素空间扩散生成。具体而言，模型以较大的 patch 尺寸（如 $16\times16$）划分输入图像，每个 patch 对应扩散变换器中的一个 token，保持了与潜在扩散模型相近的序列长度，从而使计算代价可控。与此同时，模型不再以简单的线性投影将 transformer 隐状态直接映射为像素值，而是在每个 patch 位置上动态生成一个两层 MLP 的权重矩阵，以神经场的方式在 patch 内的坐标系上解码像素级速度场（pixel-level velocity field）。数学上，给定第 $n$ 个 patch 对应的 transformer 隐状态 $X^n$，模型通过线性层预测出 MLP 的权重：

$$W_1^n,\, W_2^n = \mathrm{Linear}(\mathrm{SiLU}(X^n))$$

对于 patch 内坐标 $(i,j)$ 处的像素，首先利用离散余弦变换（DCT）基函数对坐标进行编码，得到位置嵌入 $\phi(i,j)$；再将其与该位置的含噪像素值拼接，送入由上述权重矩阵参数化的两层归一化 MLP，最终输出该像素的速度预测值 $v_{i,j}^n$。整个流程可以紧凑地写成：

$$v_{i,j}^n = W_2^n \cdot \mathrm{RMSNorm}\!\left(W_1^n \cdot [\phi(i,j) \,\|\, x_{i,j}^{\text{noisy}}]\right)$$

坐标编码选择 DCT 基函数而非正弦/余弦编码，消融实验表明这一选择对最终 FID 有显著影响，因为 DCT 基在有限离散域上具有更好的能量集中特性，能更高效地表达空间结构。整个扩散框架采用整流流（rectified flow）范式进行速度预测训练。除了主要的扩散损失外，训练中还引入了一个辅助的表征对齐损失，权重设为 0.5，将中间特征与预训练 DINOv2-Base 模型的特征空间对齐，以改善生成图像的语义一致性与细节质量。模型的其他现代化组件包括：SwiGLU 激活函数、RMSNorm 归一化，以及旋转位置编码（RoPE）。

在实验设计方面，作者首先在 ImageNet 类别条件生成任务上进行了系统评估，分别针对 $256\times256$ 和 $512\times512$ 两种分辨率进行了大量消融实验与基线对比。在 ImageNet $256\times256$ 上，PixNerd 使用 100 步 Euler 采样获得了 2.15 的 FID，在不依赖 VAE 的像素扩散方法中处于领先水平；而最优配置下（结合分类器自由引导，CFG）FID 进一步降至 1.93，达到了与当前主流潜在扩散模型竞争的性能水平。在 $512\times512$ 分辨率上，FID 为 2.84，同样无需任何级联超分模块。推理效率方面，得益于大 patch 设计（序列长度大幅缩短）以及单阶段推理路径，PixNerd 相比同类像素扩散方法（如 PixelFlow、ADM）实现了近 $8\times$ 的延迟降低。消融实验还系统验证了 DCT 坐标编码、神经场层数与通道宽度（2 层 64 通道为最优平衡点）、表征对齐损失等各组件对性能的贡献。此外，作者将 PixNerd 扩展到文本到图像生成任务，在约 4500 万张开源图像上训练 PixNerd-XXL/16 变体，在 GenEval 评测基准上取得 0.73 的综合得分，在 DPG 评测基准上取得 80.9 的平均得分，验证了该框架在大规模开放域文本引导生成场景下的竞争力。一个值得关注的额外能力是任意分辨率无训练生成：由于神经场解码本质上是基于坐标的连续函数，在推理时可以通过插值坐标编码来处理训练时未见过的分辨率，同时保持 token 数量不变，这为实际部署提供了灵活性。

PixNerd 的研究意义在于为像素空间图像生成开辟了一条兼顾质量与效率的新路径。长期以来，像素空间扩散模型被认为必须以高计算代价为前提，而潜在扩散模型虽然高效却受限于 VAE 的重建质量天花板。PixNerd 通过将隐式神经场引入扩散变换器的解码环节，优雅地解决了"大 patch 致使空间细节丢失"的矛盾，使得单阶段像素扩散在 FID 和推理速度上均能媲美乃至超越复杂的两阶段或级联方法。这一思路揭示了神经场在生成建模中的新用途，不再仅用于表示单个场景或物体，而是作为通用的局部解码器嵌入扩散框架；同时也表明，放弃 VAE 的两阶段范式并非一定要以效率换质量，精心设计的单阶段像素方案同样可以达到业界前沿水平。对于扩散模型架构研究、像素空间生成建模以及高分辨率文本到图像合成等方向，PixNerd 都提供了重要的设计参考与理论启示。
