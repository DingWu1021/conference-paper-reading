# Context parroting: A simple but tough-to-beat baseline for foundation models in scientific machine learning

> **Venue**: ICLR 2026 | **ArXiv**: 2505.11349 | **ContextParroting**

近年来，以Chronos、TimesFM和TimeMoE为代表的时间序列基础模型（time-series foundation models）在零样本预测（zero-shot forecasting）领域取得了令人瞩目的成绩。这些模型通过在海量时间序列数据上进行大规模预训练，声称能够在不了解目标系统底层物理机制的情况下，仅凭一段短暂的上下文轨迹便对未来状态做出准确预测。科学机器学习（scientific machine learning）领域对此寄予厚望，期待这些大规模基础模型能够真正学习到动力系统的内在规律，从而实现跨系统的泛化能力。然而，本文作者Yuanzhao Zhang与William Gilpin却对此提出了根本性的质疑：这些模型的优异表现究竟来自于对物理规律的真正理解，还是仅仅来自于对输入上下文的机械复制？

本文的核心贡献在于提出并分析了"上下文鹦鹉学舌"（context parroting）这一极简基线方法，并证明其性能在多类动力系统预测任务上显著超越了当前最先进的时间序列基础模型。上下文鹦鹉学舌的工作原理直观而简洁：给定一段长度为 $L$ 的历史轨迹作为上下文，算法首先从上下文末尾截取长度为 $D$ 的查询片段作为"查询动机"（query motif），随后在历史轨迹的更早部分中扫描寻找与该查询动机最相似的历史片段，最后将找到的最佳匹配片段之后的 $H$ 个时间步直接复制作为未来预测结果。整个过程不需要任何可训练参数，也无需任何形式的模型训练。

从数学上看，这一方法可被形式化为Nadaraya-Watson核估计器。设查询向量为 $q$，上下文中各历史片段为 $\{x_i\}$，预测输出为：

$$\hat{y}(q) = \sum_i w(q, x_i) \cdot x_{i+1:i+H}$$

其中权重 $w(q, x_i)$ 随查询与历史片段距离的增大而减小。在带宽趋近于零的极限情况下，该方法退化为纯粹的1近邻匹配。更一般地，连续形式的条件概率估计为：

$$\hat{p}(y | q) = \frac{\sum_i K_\sigma(q, x_i) K_\sigma(y, x_i)}{\sum_i K_\sigma(q, x_i)}$$

其中高斯核函数为 $K_\sigma(u, v) = (2\pi\sigma^2)^{-d/2} \exp\left(-\frac{\|u - v\|^2}{2\sigma^2}\right)$，$d$ 为嵌入维度与数据维度的乘积。

在理论层面，作者还深入探讨了上下文鹦鹉学舌与语言模型中"归纳头"（induction heads）机制之间的深刻联系。归纳头是Transformer在预训练过程中自然涌现的神经回路，其核心功能正是识别上下文中的重复模式，并复制该模式之后出现的内容。这一机制与上下文鹦鹉学舌的工作原理高度吻合：二者都依赖于在上下文中定位匹配片段并复制后续序列。这一发现为近期大量研究中将文本预训练的大型语言模型直接迁移用于时间序列预测的"令人费解的有效性"提供了理论解释——语言模型在预训练中习得的归纳头机制自然地泛化到了时间序列场景。

另一重要理论贡献是将上下文鹦鹉学舌的预测精度与混沌吸引子的分形维度联系起来。作者证明，随着上下文长度 $L$ 的增加，查询动机与历史片段之间的最近邻距离以代数方式衰减：

$$\ell \propto L^{-\alpha}$$

其中衰减指数 $\alpha$ 与混沌吸引子的关联维度 $d_{\text{cor}}$ 直接相关：

$$\alpha = \frac{1}{d_{\text{cor}}}$$

例如，对于二维吸引子，距离随上下文长度的增大以 $1/\sqrt{L}$ 的速率衰减。这一理论预测在135个混沌动力系统上得到了强有力的实验验证（Spearman相关系数 $r = 0.44 \pm 0.08$），揭示了上下文长度与预测精度之间类神经缩放律（in-context scaling law）的几何本质。

在实验设计方面，作者构建了全面的评测体系。他们使用包含135个混沌动力系统的dysts数据集，这些系统的维度从3维到6维不等，涵盖低维混沌吸引子、湍流、耦合振荡器以及心电图等多类真实物理系统。对每个系统，作者生成长度为 $10^5$ 的长轨迹，随机选取20个初始条件，预测未来300个时间步（约10个李雅普诺夫时间）。评测指标采用对称平均绝对百分比误差（symmetric MAPE）和有效预测时间（Valid Prediction Time）。参与对比的基础模型包括Chronos（约200M参数）、TimesFM-2.0（500M参数）和TimeMoE（200M参数）。

实验结果令人震惊：上下文鹦鹉学舌在几乎所有动力系统类型上均超越了上述所有大规模基础模型，而其计算成本仅为这些模型的五到六个数量级分之一。深入分析揭示，当基础模型表现相对较好时，其预测轨迹与上下文鹦鹉学舌的输出高度相似，表明这些模型本身也在执行某种形式的复制策略。当基础模型偏离鹦鹉学舌行为时，它们反而表现出收敛于均值（converging to the mean）等典型失效模式。值得注意的是，Chronos在上下文极短（512个token）时表现略优于鹦鹉学舌基线，暗示其可能在极端短上下文条件下习得了超越简单复制的额外策略；然而随着上下文长度增加，Chronos的性能迅速饱和，而鹦鹉学舌的性能则持续提升，二者差距不断扩大。

本文的意义远超一个简单的负面结果（negative result）。它对当前时间序列基础模型在科学机器学习中的定位提出了深刻的方法论反思：现有模型是否真正学习到了物理系统的动力学规律，还是仅仅成为了精心包装的上下文复制机器？这一发问对于推动科学机器学习领域的健康发展具有重要价值。作者并非否定基础模型的所有价值，而是强调：在宣称基础模型具有某种物理理解能力之前，必须首先以上下文鹦鹉学舌这一强力基线为参照，否则评估结论将严重失真。从更深远的角度看，本文揭示的分形维度与预测精度之间的理论联系，为理解时间序列基础模型的内在机制开辟了新的研究路径，也为设计真正超越鹦鹉学舌、能够内化物理规律的下一代科学机器学习模型指明了方向。
