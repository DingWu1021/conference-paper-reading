# Learning to See Before Seeing: Demystifying LLM Visual Priors from Language Pre-training

> **Venue**: ICLR 2026 | **ArXiv**: 2509.26625 | **LLMVisualPriors**

大型语言模型（LLM）在仅接受纯文本训练的情况下，竟然能够发展出丰富的视觉理解能力——这一现象长期以来被视为多模态学习领域中令人困惑却又引人入胜的谜题。现有的多模态大模型（MLLM）构建范式通常假设视觉能力完全来自视觉对齐阶段或视觉指令微调阶段，而语言预训练的贡献往往被忽视或简单化处理。这篇发表于 ICLR 2026（口头报告）的论文，由 Junlin Han、Shengbang Tong、David Fan 等七位作者共同完成，系统性地揭示了语言预训练过程中所形成的"视觉先验"的内在结构与来源，为下一代多模态大模型的数据驱动设计提供了坚实的实证基础。

研究的核心出发点在于：LLM 在语言预训练阶段究竟习得了哪些隐式的视觉世界知识，这些知识如何在后续视觉对齐中被激活，以及不同类型的预训练数据对视觉先验的贡献是否存在本质差异。作者将语言预训练阶段所产生的隐式视觉知识统称为"视觉先验"（visual priors），并提出这一概念可以进一步被分解为两个相互独立的子成分：**感知先验**（perception prior）与**推理先验**（reasoning prior）。感知先验刻画了模型对视觉世界中物体、场景等具体对象的低层次表征能力，而推理先验则对应于在视觉问答、跨模态推理等高层认知任务上的表现。两者不仅来源不同，在规模缩放规律（scaling law）上也表现出截然不同的行为特征。

在方法论层面，研究团队构建了一套极为系统的受控实验框架，覆盖了从 LLM 预训练到视觉对齐、再到多模态监督微调的完整 MLLM 构建流程。实验共消耗超过 50 万 GPU 小时，跨越五个模型规模（340M 至 13B 参数），并对 16 类预训练数据类别进行了精细的消融分析。预训练骨干网络采用 Llama-3 架构，优化器为 AdamW，视觉对齐阶段使用 MLP 投影层将视觉编码器的表征映射至语言模型的输入空间，之后再进行多模态监督微调。评估体系涵盖 16 个多模态基准，分为通用 VQA（如 GQA、MME、MMBench、SEED）、知识密集型 VQA（如 ScienceQA、MMMU、AI2D、MathVista）、OCR 与图表理解（TextVQA、ChartQA、OCRBench），以及以视觉为核心的任务（RealWorldQA、Blink、COCO 分割、ADE 场景理解、Omni3D 三维理解）。此外，研究团队还专门构建了一个新型基准 **MLE-Bench**（Multi-Level Existence Bench），包含 1861 张图像的四选一物体存在性问答题，按照目标占图像像素比例分为小（0-30%）、中（30-60%）、大（60-100%）三个层次，专用于评估感知先验在不同尺度下的表现。

在核心发现方面，论文揭示了以下几个相互关联的重要规律。首先，**推理先验主要源自推理密集型数据**，包括代码、数学和学术文献等内容，这些数据能够显著提升 LLM 在视觉推理任务上的表现，且随着此类数据比例（从 0% 到 75%）的增加呈现出持续进步的趋势；而超过 75% 之后，性能增益趋于平稳乃至略有下降。相比之下，**感知先验则更为弥散地涌现于多样化的广域语料**中，其提升主要依赖于视觉对齐阶段和视觉指令微调中的数据质量与规模，而非语言预训练中的特定数据类别。其次，**描述视觉世界的文本数据**（如图像说明、视觉描述）虽然对感知能力有所贡献，但其性能提升效果随着占比增加迅速饱和，并不如推理数据那样具有持续的边际效益。第三，通过互最近邻（mutual nearest-neighbor，mNN）评分衡量 LLM 表征与视觉编码器表征之间的核相似度，研究发现推理数据比例与对齐分数呈现非单调关系，在约 75% 时达到峰值，这与视觉推理性能的规律高度一致，暗示**结构化文本（代码、数学）与视觉数据存在深层次的构成相似性**，有助于跨模态表征的自然对齐。

在假设验证与扩展分析方面，作者提出并验证了三个核心假设。其一，感知先验具有多层次性（Multi-Level Perception Prior）：在 MLE-Bench 上，多样化的网络抓取数据对小中尺寸物体的检测表现出显著优势，而这一优势随物体尺寸增大逐渐消失，说明感知先验在不同粒度的视觉识别任务上表现不一。其二，推理先验具有跨模态可迁移性（Cross-Modal Reasoning Transfer）：将代码数据占比从 0% 提升至 100% 时，视觉推理中的逻辑合理性从 4.52% 上升至 9.52%，推理深度提升近六倍，表明语言推理能力是一种与模态无关的通用认知能力，可直接迁移至视觉推理场景。其三，结构化文本与视觉数据在表示空间上具有结构对齐性（Structural Alignment），进一步支持了代码和数学训练有益于视觉感知的机制解释。

基于上述发现，研究团队提出了一套以数据为中心的视觉感知 LLM 预训练配方。最优混合比例大致为：约 60% 的推理相关数据（代码、数学、学术）与约 15% 的视觉描述文本，其余为通用语料。这一配方在 1 万亿 token 规模的预训练中得到了验证：使用"均衡配方"（mix6）训练的 7B 模型相比"语言优化配方"（mix0）在 VQA 任务上从 37.32% 提升至 38.64%，同时在语言困惑度上也优于对比基线（7.49 vs. 8.72）。此外，作者还提出了一种有趣的技术——"盲视觉指令微调"（Blind Visual Instruction Tuning），即在完全不给模型提供图像的情况下进行视觉指令微调，借助模型已有的视觉先验实现对视觉任务的部分解题，进一步印证了语言预训练所形成的隐式视觉知识的真实存在与可激活性。

从更宏观的角度审视，这项工作深刻响应了 Platonic Representation Hypothesis 的核心论断：足够大规模的模型在不同训练模态下会自然收敛至统一的世界模型表征空间。通过清晰地区分感知先验与推理先验，并追溯两者的数据来源与缩放行为，本文将多模态大模型的构建从依赖偶发性规模扩展的"经验主义"路径，推进至以数据配方为核心的"战略性"设计范式。对于研究者而言，这意味着不必再盲目增大全域数据规模，而是可以针对性地调整预训练语料的类别构成，以更高效的方式激活和增强 LLM 的视觉先验。这一思路对于资源受限场景下的多模态模型开发尤具实践价值，也为探索语言与视觉认知之间的深层关联提供了重要的实证支撑。
