# AgentGym-RL: An Open-Source Framework to Train LLM Agents for Long-Horizon Decision Making via Multi-Turn RL

> **Venue**: ICLR 2026 | **ArXiv**: 2509.08755 | **AgentGymRL**

大语言模型（LLM）在单轮问答、代码生成等静态任务上已展现出令人印象深刻的能力，但如何让 LLM 在动态、多步骤、开放式环境中自主决策，仍是智能体研究领域的核心难题。现实世界中的智能体任务往往需要跨越数十乃至数百个交互步骤才能完成目标，例如在网络上自动完成机票预订、跨多个知识源进行多跳推理检索，或是在模拟物理环境中完成操纵任务。这类长视野决策问题（long-horizon decision making）不仅要求模型具备扎实的基础推理能力，还必须在探索与利用之间保持精妙的平衡——过于保守则难以发现有效的解题路径，过于激进则导致训练崩溃。与此同时，学术界长期缺乏一个统一的、可交互的强化学习训练框架，能够同时支持多种环境、多种 RL 算法以及灵活可扩展的智能体模块。正是在这一背景下，来自复旦大学等机构的研究团队提出了 AgentGym-RL，这一工作于2025年9月以预印本形式发布，并最终获得 ICLR 2026 口头报告（Oral）的认可。

AgentGym-RL 框架在架构层面采用高度模块化、解耦的设计思想，将系统划分为三大核心组件：环境模块（Environment Module）、智能体模块（Agent Module）与训练模块（Training Module）。环境模块通过标准化的服务器-客户端架构（基于 HTTP 协议）提供多样化的任务场景，涵盖网页导航、深度搜索、数字游戏、具身任务与科学探索五大类别，共计27个任务，从而为智能体提供了丰富、现实且具有挑战性的训练土壤。智能体模块封装了推理与多轮交互决策逻辑，支持长视野规划、自我反思等高阶认知机制。训练模块则实现了包括 PPO、GRPO、REINFORCE++ 以及 RLOO 在内的主流强化学习算法管道，确保研究者可以灵活切换和比较不同的训练策略。

在方法核心贡献方面，论文提出了 ScalingInter-RL 这一渐进式交互视野扩展策略，专门针对长视野 RL 训练中的不稳定性问题而设计。研究者发现，若在训练早期就给予模型过大的交互预算（即允许进行过多步骤的交互），模型往往会产生大量冗余推理和无效动作，最终导致训练崩溃（training collapse）；但过于严格的交互限制又会压制探索多样性，使模型局限于简单策略而无法习得复杂行为。ScalingInter-RL 通过引入单调递增的交互步数上限调度方案 $\{h_1 < h_2 < \cdots < h_n\}$ 来解决这一两难困境，其中 $h_t$ 定义了训练阶段 $t$ 时允许的最大交互轮次。在训练初期，较小的 $h_1$ 迫使智能体专注于利用（exploitation），掌握基础技能并解决相对简单的任务；随着训练进入中后期，逐渐放大的 $h_t$ 鼓励更广泛的探索（exploration），引导模型发展出多样化的问题解决策略，并逐步挑战更复杂的长视野任务。这种从易到难的课程式设计不仅在工程层面稳定了训练过程，更在智能体行为层面催生了规划、反思和策略性回溯等高阶认知行为的涌现。

在实验设计方面，研究团队以 Qwen2.5-3B 和 Qwen2.5-7B 作为骨干模型，在上述27个多样化任务上开展了系统性评估，并与 OpenAI o3、Gemini-2.5-Pro、GPT-4o 等顶级商业模型以及多种开源基线进行对比。实验结果令人瞩目：经过 RL 训练后，7B 参数规模的开源模型平均提升了 33.65 分，最终在整体性能上与 o3 等商业旗舰模型持平甚至超越。在具体任务层面，ScalingInter-7B 在 SciWorld 科学探索任务中取得 57.00 分，而 o3 仅为 41.50 分；在 TextCraft 文字合成游戏中获得 91.00 分，达到与顶级专有模型相当的水准；在 BabyAI 具身导航任务上以 96.67% 的准确率刷新了当前最优记录；在深度多跳搜索任务中以 38.25 分超越 GPT-4o 的 26.75 分。这些数据充分说明，在合理的训练框架和策略下，小参数量的开源模型具备挑战大规模商业模型的能力。此外，研究还揭示了一个重要规律：训练后处理（post-training）与测试时计算（test-time compute）所体现出的扩展潜力，远高于单纯扩大模型参数规模的效益——7B 模型在经过充分 RL 训练后，整体表现甚至优于未经 RL 优化的 72B 基线模型，这为未来研究方向提供了极具价值的指引。

从更宏观的视角审视，AgentGym-RL 的意义不仅仅在于提出了一个性能优越的训练方法，更在于它以全面开源的姿态（代码、数据集、模型权重均对外发布）填补了智能体 RL 研究领域长期存在的基础设施空白。统一、模块化的框架降低了研究者复现和比较不同方法的门槛，使得社区可以在同一基准下公平评估各类算法和策略的优劣。ScalingInter-RL 所揭示的渐进式视野扩展思想，与课程学习（curriculum learning）领域的经典理论一脉相承，但将其系统化地引入多轮交互 RL 训练场景，并以实验证明其在防止崩溃、促进行为涌现方面的有效性，具有重要的原创价值。随着 LLM 智能体被逐步部署于更复杂、更开放的现实应用场景，如何在长视野交互中维持稳定高效的学习动态，将成为智能体研究的关键议题，而 AgentGym-RL 为这一方向提供了坚实的基础工具与有益的研究范式。
