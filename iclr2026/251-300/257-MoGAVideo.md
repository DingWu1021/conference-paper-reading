# MoGA: Mixture-of-Groups Attention for End-to-End Long Video Generation

> **Venue**: ICLR 2026 | **ArXiv**: 2510.18692 | **MoGAVideo**

长视频生成是近年来视频生成领域最具挑战性的课题之一。随着扩散变换器（Diffusion Transformers，DiTs）在图像与短视频合成上取得显著突破，研究者开始尝试将其扩展至分钟级的长视频生成。然而，DiTs 的核心瓶颈在于全注意力（Full Attention）机制随序列长度呈平方级增长的计算复杂度。以 480p、24fps、一分钟时长的视频为例，其对应的视觉 token 数量约达 58 万，在此规模下进行完整的注意力计算已在实践中几乎不可行。已有部分工作尝试通过分块稀疏注意力（blockwise sparse attention）来缓解这一问题，但块级别的粗粒度估计在精度与效率之间存在固有的权衡困境——块尺寸越大，索引到的相关 token 越不准确；块尺寸越小，则稀疏化收益有限。此外，现有稀疏方案往往依赖自定义 CUDA 核函数，与 FlashAttention、序列并行等现代高效注意力框架难以兼容，工程落地门槛较高。基于以上背景，来自中国科学技术大学、字节跳动番茄 AI、香港科技大学及武汉大学的研究团队提出了 MoGA（Mixture-of-Groups Attention），一种精确的、可学习的稀疏注意力机制，并将其应用于端到端分钟级长视频生成任务，取得了令人瞩目的成果。

MoGA 的核心思想是：注意力机制本身具有高度冗余性，每个 query 的输出主要由少数最相关的 key-value 对决定。因此，可以通过一个轻量可学习的 token 路由器（token router）来精确筛选出对每个 query 最重要的 token，而无需依赖块级别的粗粒度近似。具体而言，路由器对输入序列 $x$ 计算路由分数 $r = \text{Router}(x)$，再经 softmax 归一化得到每个 token 对 $M$ 个组的归属概率 $p(i|x) = \text{softmax}(r)_i$，最终每个 token 被分配到得分最高的组 $g(x) = \arg\max_i p(i|x)$。注意力计算则在各组内独立进行，输出为 $\text{MoGA}(x) = p(g(x)|x) \cdot \text{SA}(q,\, K_{g(x)},\, V_{g(x)})$，其中 $\text{SA}$ 表示标准点积注意力。若将序列划分为 $M$ 个组，注意力计算的整体复杂度从 $O(N^2)$ 降低至 $O(N^2/M)$，当 $M=5$ 时即可实现约 71% 的 FLOPs 稀疏率。

为防止路由器退化（即所有 token 集中涌入少数几个组），研究者引入了一个辅助的组平衡损失（Group Balancing Loss）：

$$\mathcal{L}_{gb} = \alpha \cdot M \sum_i F_i P_i$$

其中 $F_i$ 为分配至第 $i$ 组的 token 占比，$P_i$ 为所有 token 在第 $i$ 组上的平均路由概率，损失权重 $\alpha=0.1$。该正则项鼓励各组接收到均匀的 token 流量，从而充分发挥每个组的语义聚合能力，并维持全局信息多样性。

在架构层面，MoGA 被整合进基于 DiT 的视频生成框架，并与空间-时序组注意力（Spatial-Temporal Group Attention，STGA）协同工作。STGA 采用固定的时空窗口划分，专注于捕获局部连续性（如相邻帧之间的运动一致性），而 MoGA 则通过语义路由补充长程跨帧依赖。两者分工互补，共同构成高效且表达力强的注意力组合。此外，MoGA 采用无自定义核函数（kernel-free）的设计，注意力计算仍遵循标准点积形式，因此可以直接与 FlashAttention 和序列并行（sequence parallelism）无缝结合，大幅降低了工程复杂度。为支持多镜头（multi-shot）视频生成，模型还设计了镜头级文本条件机制，通过多模态注意力层将每个镜头的文本描述注入对应时间段的视觉 token，实现了细粒度的跨镜头语义控制。

为训练该模型，研究团队构建了一个专门的分钟级多镜头训练数据流水线。流水线分为两个阶段：第一阶段在视频层面进行质量过滤、镜头分割（结合 AutoShot 与 PySceneDetect），第二阶段在镜头层面进行 VQA 过滤、水印去除、多模态字幕生成及片段合并（最长合并至 65 秒）。最终得到的训练数据集包含时序连贯、语义丰富的分钟级多镜头视频样本。模型以 Wan2.1（1.3B 和 14B 参数版本）为基础进行微调，采用多阶段训练策略：先以 10 秒片段训练 3000 步，再以 30 秒片段训练 1000 步，学习率设为 $1\times10^{-5}$，组数 $M=5$（Wan 架构）或 $M=20$（MMDiT 架构）。

在定量评估方面，MoGA 在主体一致性（Subject Consistency）、背景一致性（Background Consistency）、运动平滑性（Motion Smoothness）、美学质量（Aesthetic Quality）及跨镜头 CLIP/DINO 相似度等多项指标上均优于或持平于全注意力基线。以 Wan2.1-14B 为基础的对比实验表明，全注意力（0% 稀疏率）下主体一致性得分为 0.9611，计算量为 6.94 PFLOPs；而 MoGA（M=5，稀疏率 71.25%）不仅将计算量压缩至 2.26 PFLOPs（降低约 67%），主体一致性得分反而提升至 0.9699，体现了语义感知路由在减少冗余计算的同时有效聚焦了关键语义信息。在端到端生成能力上，系统可在约 58 万 token 的上下文长度下生成 480p、24fps、分钟级多镜头视频，涵盖场景切换、镜头衔接等复杂叙事结构，充分展示了 MoGA 在超长序列建模中的实际有效性。

综合来看，MoGA 在长视频生成领域提出了一个兼具精度与效率的稀疏注意力范式。其核心创新在于将 Mixture-of-Experts 式的路由思想引入注意力 token 分组，以可学习的语义路由取代启发式的块级近似，在大幅节省计算开销的同时保留甚至提升了模型对长程依赖的建模能力。无需自定义算子的设计哲学使其具备极强的工程可迁移性，能够直接受益于 FlashAttention 等主流高效注意力实现。数据构建侧的多阶段多镜头流水线为端到端分钟级视频生成提供了高质量的训练基础。未来，随着视频生成向更高分辨率、更长时长的方向演进，MoGA 所代表的语义感知稀疏注意力技术路线有望成为该领域的重要基础设施，并为其他需要处理超长序列的多模态任务（如长文档理解、音视频联合生成等）提供方法论借鉴。
