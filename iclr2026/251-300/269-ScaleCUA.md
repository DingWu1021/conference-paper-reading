# ScaleCUA: Scaling Open-Source Computer Use Agents with Cross-Platform Data

> **Venue**: ICLR 2026 Oral | **ArXiv**: 2509.15221 | **ScaleCUA**

近年来，以视觉语言模型（Vision-Language Models，VLMs）为核心驱动力的计算机使用智能体（Computer Use Agents，CUAs）迅速成为人工智能领域的研究热点。这类智能体能够自主理解屏幕截图所呈现的图形用户界面（GUI），并通过模拟鼠标点击、键盘输入、滚动等操作完成复杂的自动化任务，其应用潜力覆盖了从网页浏览、文档编辑到移动端应用操作的广泛场景。然而，与自然语言处理或图像识别领域相比，计算机使用智能体的研究面临一个根本性的数据瓶颈：高质量的计算机操作轨迹数据极为稀缺且代价高昂。人工标注完整的操作序列不仅费时费力，还需要标注者具备相当的专业背景，这导致现有开源社区中能够支持大规模训练的数据集极为有限。与此同时，商业闭源模型（如 Anthropic 的 Claude Computer Use、OpenAI 的 Operator 等）在大量私有数据上完成了充分的训练，形成了对开源方案的显著优势。正是在这一背景下，ScaleCUA 应运而生，旨在通过构建跨平台大规模数据集并开源相应的基础模型，填补开源计算机使用智能体领域的空白。

ScaleCUA 的核心技术贡献体现在三个紧密衔接的层面。首先，在数据构建层面，研究团队提出了一套"闭环流水线"（closed-loop pipeline）方法，将自动化智能体与人类专家的协同判断有机结合。该流水线的运作逻辑是：先由自动化系统在受控环境中批量生成候选操作轨迹，再由人类专家对轨迹的正确性和完整性进行审核与校正，最终形成高质量、大规模的训练数据。这种人机协同的数据生产范式既保证了数据的规模效应，又通过人类监督确保了轨迹的质量上限，有效解决了纯自动化收集容易产生噪声的问题。所构建的 ScaleCUA-Data 数据集横跨 6 个操作系统平台（包括 Windows、macOS、Ubuntu、Android 等）以及 3 个任务领域，是目前开源社区中规模最大、平台覆盖最广的计算机使用轨迹数据集之一。其次，在模型设计层面，ScaleCUA 提出了一种双模式架构：一是"原生智能体模型"（Native Agentic Model），由单一视觉语言模型同时承担 UI 元素定位（grounding）与任务规划（planning）两项功能，具有结构简洁、推理高效的优势；二是"智能体工作流"（Agentic Workflow），将定位与规划分解为两个独立的专用模型，通过模块化的协作机制提升复杂任务下的整体性能。这种灵活的双模式设计使得用户可以根据计算资源和任务复杂度自由选择适合的部署方案。ScaleCUA 在模型基座选择上支持两种主流开源视觉语言模型：Qwen2.5-VL 和 InternVL，并通过 vLLM 进行模型部署与推理，兼容 OpenAI 风格的 API 接口，极大降低了研究者的使用门槛。第三，在训练策略层面，研究通过大规模数据的监督微调（Supervised Fine-Tuning，SFT）完成模型训练，将跨平台操作能力内化为模型的基础能力，使得同一模型无需针对特定平台进行额外适配即可实现跨平台泛化操作。

在实验设计上，ScaleCUA 构建了覆盖所有主要操作系统的全面评测体系，充分体现了其跨平台定位。评测基准包括：Android 平台上的 AndroidWorld 和 AndroidLab，Ubuntu 平台上的 OSWorld，macOS 平台上的 MacOSArena，Windows 平台上的 WindowsAgentArena，以及面向网页操作的 WebArena-Lite-v2。此外，为了更全面地评估 GUI 定位与理解能力，还引入了 ScreenSpot-Pro 和 MMBench-GUI 等专项评测集。在定量结果上，ScaleCUA 取得了一系列令人瞩目的成绩。在 WebArena-Lite-v2 上，ScaleCUA 达到 47.4% 的成功率，相较基线模型提升了 26.6 个百分点，刷新了该基准的最优记录。在 ScreenSpot-Pro 定位基准上，ScaleCUA 相较基线提升了 10.7 个百分点，显示出模型在精确 UI 元素定位方面的显著进步。在 MMBench-GUI L1-Hard 难度级别上，ScaleCUA 取得了 94.4% 的准确率，接近人类表现上界；在 OSWorld-G 任务完成基准上取得了 60.6% 的成功率。这些结果在开源模型中均达到了当前最优水平（state-of-the-art），并在多个任务上缩小了与商业闭源模型的差距。值得关注的是，ScaleCUA 在跨平台评测中展现出一致的强劲表现，而非仅在特定平台上的局部优化，这印证了跨平台数据联合训练所带来的泛化收益。消融实验进一步验证了数据规模与平台多样性对最终性能的关键贡献，随着数据量增加，各平台上的任务完成率均呈现出明显的提升趋势，符合"规模定律"（scaling law）的基本预期。

综合来看，ScaleCUA 是开源计算机使用智能体领域的一项重要进展。其研究意义体现在多个维度：在实践层面，ScaleCUA 通过开源数据集、模型权重和训练代码，为学术界和工业界提供了构建高性能 GUI 自动化系统的完整基础设施，有效降低了进入该领域的门槛；在方法论层面，人机协同的闭环数据构建范式为其他高成本标注任务提供了可借鉴的范式——将自动化生产的规模优势与人类监督的质量保障相结合，是一种在数据效率上较为实用的解法；在研究影响层面，ScaleCUA 作为 ICLR 2026 的口头报告（Oral）论文，其工作推动了开源生态向商业模型的追赶步伐，可能进一步激励社区在更多平台和任务领域上持续扩充高质量的计算机使用轨迹数据。未来，随着视觉语言模型底座能力的持续提升、强化学习（RLHF/RLVR）等后训练技术的引入，以及更精细的平台专属数据的补充，ScaleCUA 所构建的这一技术框架有望催生真正意义上的通用桌面/移动端自动化智能体，从而为人机交互方式带来深远的变革。
