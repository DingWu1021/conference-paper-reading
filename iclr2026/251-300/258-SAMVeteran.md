# SAM-Veteran: An MLLM-Based Human-like SAM Agent for Reasoning Segmentation

> **Venue**: ICLR 2026 | **ArXiv**: N/A | **SAMVeteran**

推理分割（Reasoning Segmentation）是视觉-语言理解领域中极具挑战性的任务，其目标是根据复杂的自然语言查询（例如隐含属性描述、多步骤推理要求）精确地定位并分割图像中的目标区域。与传统的指代表达分割（Referring Expression Segmentation）不同，推理分割要求模型不仅能理解语义，还必须完成多跳推理、常识推断等高阶认知过程，最终输出像素级别的精准掩码。Segment Anything Model（SAM）的出现为通用分割任务带来了革命性突破：SAM 以其强大的零样本泛化能力和交互式分割机制（支持边界框、点击点等多种提示方式）迅速成为分割领域的基础模型。然而，现有的将多模态大语言模型（MLLM）与 SAM 相结合的方法普遍存在一个显著缺陷——它们未能充分利用 SAM 的迭代精化能力。现有范式通常仅将 SAM 作为一个"一次性"的分割后端：MLLM 生成提示（如边界框），SAM 据此生成掩码，整个流程就此终止。然而，人类用户在使用 SAM 时会自然地观察初始分割结果，判断其质量，并通过提供额外的正负点击点来迭代精化掩码，直到满意为止。这种人类专家级别的交互模式在现有自动化方法中几乎完全缺失，导致分割质量的天花板远低于 SAM 的真实潜力。

正是针对上述问题，本文提出了 SAM-Veteran 系统。其核心设计理念是：将多模态大语言模型训练成一个能够模仿资深人类用户与 SAM 交互方式的智能代理（Agent），通过有推理依据的多轮交互工作流来完成推理分割任务。SAM-Veteran 的整体工作流程由三个紧密耦合的步骤组成。第一步是边界框生成（Bounding Box Generation）：给定输入图像与自然语言查询，MLLM 基于对图像内容的理解与对查询的深度推理，预测目标区域的边界框坐标，并将其作为初始提示输入 SAM，得到初始分割掩码。第二步是精化点提议（Refinement Point Proposal）：MLLM 接收原始图像、查询以及 SAM 生成的当前掩码作为输入，综合分析掩码的分割质量——识别哪些区域被错误地包含（假正例）或遗漏（假负例）——并据此提议新的正点击（foreground points）或负点击（background points），以引导 SAM 对掩码进行精化。第三步是自适应终止（Adaptive Termination）：MLLM 在每轮精化后评估当前掩码是否已达到满意质量，若是则停止迭代，否则继续提议新的精化点，直至满足终止条件。这三步共同构成了一个闭环的、可感知掩码质量的交互式分割智能体，与人类专家操作 SAM 的行为模式高度一致。

在训练框架层面，SAM-Veteran 采用了基于群体相对策略优化（Group Relative Policy Optimization，GRPO）的多任务强化学习框架。GRPO 是强化学习中策略优化方法的一种变体，其核心思想是通过对一组采样输出进行群体相对评分来估计策略梯度，从而在无需显式价值函数的情况下完成优化。在 SAM-Veteran 的场景中，GRPO 被用于同时增强 MLLM 在两项关键能力上的表现：一是文本定位能力（Textual Grounding），即根据复杂语言描述准确预测目标区域的边界框；二是掩码理解能力（Mask Comprehension），即理解 SAM 生成的掩码内容并判断其质量、提议改进点。将这两项任务纳入统一的强化学习目标中，使得模型能够在训练过程中同步习得定位与精化所需的复合能力，两者之间形成正向促进效应：更准确的定位产生更高质量的初始掩码，而更强的掩码理解能力则指导更精准的精化操作。

在动态采样策略（Dynamic Sampling Strategy）方面，论文提出了专门针对边界框预测与精化点提议两类不同输出形式而设计的采样机制。边界框坐标的生成与点击点的提议在输出空间结构、精度要求和质量评估方式上存在本质差异，若使用统一的采样策略，训练过程往往不稳定，导致两类任务的学习难以平衡。动态采样策略根据两类任务的梯度信号和学习进度自适应地调整采样分布，有效稳定了多任务强化学习训练过程，使得模型在两类输出上都能获得充分且有效的学习信号。

从实验验证的角度来看，SAM-Veteran 在多个推理分割基准数据集上进行了系统性评估，涵盖域内（in-domain）和域外（out-of-domain）两类评测设置。域内评测衡量模型在训练分布相近数据上的分割精度，而域外评测则更严格地检验模型的泛化能力——这对于推理分割这类需要处理多样化自然语言表达的任务而言尤为重要。实验结果表明，SAM-Veteran 在两类评测设置下均建立了新的最优性能（State-of-the-Art），显著超越了此前同类方法。这一成果有力地证明了将迭代式掩码精化与自适应终止机制引入推理分割框架的有效性。与仅使用边界框提示 SAM 的单次交互方法相比，SAM-Veteran 的多轮交互机制带来了实质性的分割质量提升，尤其在目标边界复杂、语义描述隐晦或目标遮挡严重的困难样本上优势更为明显。

综合来看，SAM-Veteran 的研究意义体现在以下几个层面。首先，它重新审视了 MLLM 与 SAM 协作的范式：从此前的"单次提示—单次分割"模式转变为"多轮推理—迭代精化—自适应决策"的智能代理模式，这一转变更充分地释放了 SAM 的交互式分割潜力。其次，将强化学习（特别是 GRPO）引入多模态分割任务的训练，并设计多任务目标与动态采样策略，为未来在视觉-语言协同任务上应用 RL 训练提供了有价值的方法论参考。第三，SAM-Veteran 展示了如何让 MLLM 具备掩码感知（mask-aware）能力——即不仅能生成语言和边界框输出，还能理解和评估像素级别的分割结果，这种能力对于构建更通用的视觉理解智能体具有重要意义。从更宏观的视角来看，SAM-Veteran 所代表的"具身分割代理"（Embodied Segmentation Agent）思路与当前大语言模型驱动的智能体研究趋势高度契合，为视觉感知智能体的设计开辟了新的方向。未来的研究可以在此基础上探索更复杂的多轮视觉交互场景、更精细的奖励设计，以及将该范式扩展到视频分割、三维场景理解等更广泛的视觉任务中。
