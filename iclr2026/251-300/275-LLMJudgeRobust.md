# Doubly-Robust LLM-as-a-Judge: Externally Valid Estimation with Imperfect Personas

> **Venue**: ICLR 2026 | **ArXiv**: 2509.22957 | **LLMJudgeRobust**

随着生成式人工智能（GenAI）系统在现实场景中的大规模部署，如何对这些系统的质量进行可靠评估已成为学界和工业界共同关注的核心问题。传统的 LLM 评估范式通常依赖有限规模的人类标注者对系统输出进行评分，并以此估计系统在真实部署场景下的整体表现。然而，这种做法面临一个根本性的外部效度（external validity）挑战：实验室中参与评标的标注者群体，往往在人口统计学特征、使用习惯和文化背景上与真实用户群体存在系统性偏差；与此同时，评估阶段所使用的系统输出样本也未必能代表上线后实际的输入分布。这种双重分布偏移（evaluation sampling bias）使得基于现有评估数据得到的系统质量估计值，在推广到真实部署条件时可靠性存疑，严重限制了评估结论的外部可迁移性。

近年来兴起的"LLM-as-a-Judge"范式——即让大语言模型扮演特定人口特征的"虚拟标注者"（persona）来模拟人类评分——为缓解上述问题提供了一条颇具潜力的路径。由于 LLM Persona 能够以极低成本生成大量覆盖多样化人口群体的模拟评分，理论上可以弥补真实人类标注在目标分布上的稀缺性。但现实中 LLM Persona 与真实人类评分者之间存在不可忽视的差距：模型在捕捉特定群体细粒度偏好时往往不够精确，生成的评分带有系统性偏置，相关性也并不完美。如何利用"不完美"的 Persona 评分来修正有偏的人类评分，从而得到外部有效的系统质量估计，是本文着力解决的核心问题。

本文的核心方法论贡献在于提出了一套**双重鲁棒估计框架（Doubly-Robust Estimation Framework）**。该框架的基本设定将数据建模为元组 $Z = (X, V, C, Y, \hat{Y})$，其中 $X$ 表示标注者的社会人口特征，$V$ 表示待评内容，$C$ 为标注完成指示变量，$Y$ 为真实人类评分，$\hat{Y}$ 为 LLM Persona 给出的模拟评分。框架同时刻画了两类偏移：协变量偏移（covariate shift，即源分布 $P_s$ 与目标分布 $P_t$ 在标注者和内容特征上的差异）以及选择偏差（selection bias，即标注完成率 $C$ 与标注者/内容特征相关联）。

在此设定下，论文提出的双重鲁棒估计量形式为

$$\hat{\theta} = \frac{1}{N_t}\sum_{i}\hat{\mu}(W_i^t, \hat{Y}_i^t) + \frac{1}{N_s}\sum_{j}\hat{\alpha}(W_j^s, C_j^s)\bigl\{Y_j^s - \hat{\mu}(W_j^s, \hat{Y}_j^s)\bigr\}$$

其中 $\hat{\mu}(W, \hat{Y})$ 为利用 Persona 评分和有偏源数据训练的**预测模型**（outcome model），用于预测目标分布下的人类评分期望；$\hat{\alpha}(W, C)$ 为**重加权函数**（reweighting function），用于修正协变量偏移和选择偏差。该估计量具备"双重鲁棒"性质：只要预测模型 $\hat{\mu}$ 或重加权模型 $\hat{\alpha}$ 之一以足够快的速度收敛（参数级或接近参数级速率），最终的系统质量估计即可保持统计有效性（valid coverage）。这一性质极大增强了框架在实践中的适用性——即使 Persona 评分质量有限或重加权模型存在误差，系统仍能从另一分量获得保障。对于置信区间的构造，论文同时给出了对应的方差估计量，其由预测残差项和加权平方项组合而成。

为绕开高维文本空间中密度比估计困难的问题，论文采用 **Riesz 损失**直接学习重加权函数，而非分别估计密度比和倾向得分再进行组合。实验表明，这一选择在各类实验条件下均显著优于传统的密度比拆解方法，尤其是在高维特征空间中表现最为突出。

为了系统性地评估框架在不同条件下的表现，论文构建了一套**Persona 模拟框架（Persona Simulation Framework, PSF）**，该框架通过可控的方式操纵三个关键变量：（1）协变量偏移程度，以源分布与目标分布之间的 Sinkhorn 距离度量，通过混合参数 $\zeta$ 调节；（2）选择偏差强度，通过 Beta CDF 变换模拟不同程度的标注者流失（dropout）；（3）Persona 质量，通过 Pearson 相关系数 $\rho$ 和偏置参数 $\eta$ 刻画 LLM 评分与真实人类评分的偏离程度。PSF 在三类数据集上展开实验：完全合成数据集（已知真实 nuisance 函数，用于基准对照）、基于 PRISM 数据集的半合成数据集（含 1000 条真实对话，以 GPT-5 评分作为伪真实标签）、以及基于 DICES 数据集的半合成数据集（含 300 条真实对话，每条由 25 位标注者评分）。

在与多条基线方法的系统对比实验中——基线包括朴素样本均值（Sample Average）、逆概率加权（IPW）、纯 Persona 估计、Persona 增广回归、以及近期统计学习领域的 PPI++、RePPI 等方法——论文提出的 DR (Riesz) 方法在偏差（MAE）和置信区间覆盖率两项核心指标上均表现最优。具体来看，在完全合成数据上，DR (Riesz) 达到了 1.00 的覆盖率和 0.03 的偏差；在 PRISM 半合成数据上，覆盖率为 0.93、偏差为 0.46，而对比方法 RePPI 的覆盖率仅为 0.66；在 DICES 半合成数据上，DR (Riesz) 的覆盖率为 0.86、偏差低至 0.02，而 RePPI 的覆盖率仅为 0.40。上述结果表明，双重鲁棒框架在各类协变量偏移和选择偏差条件下均能稳健维持有效的统计推断。

进一步的分析揭示了几项重要的实践规律。首先，框架对 Persona 质量具有相当强的容错性：即便 LLM 评分与人类评分的 Pearson 相关性仅为 0.43（对应实验中真实 LLM 模型的典型水平），DR (Riesz) 仍能获得与人工控制相关性至 0.6 时相当的覆盖表现，说明框架能有效从质量有限的 Persona 评分中提取有用信息。其次，在 $\rho \geq 0.65$ 时框架表现最为稳健，但即使在相关性较低的情形下，相比于忽略分布偏移的朴素估计方法，双重鲁棒框架仍具有明显优势。此外，实验还验证了以 Claude Haiku 3.5、Claude Sonnet 3.5、GPT-5 和 GPT-4o-Mini 作为 Persona 模型时框架的适用性，显示结论对不同 LLM 后端具有一定的通用性。

从更宏观的视角来看，本文的研究意义体现在多个层面。在理论上，本文首次将双重鲁棒因果推断框架引入 LLM 评估领域，系统地弥合了统计学中重加权方法与机器学习中预测增强方法之间的方法论鸿沟，并明确放宽了 PPI++ 和 RePPI 等前沿工作所依赖的"完全随机缺失"（MCAR）假设，转而支持更现实的"随机缺失"（MAR）假设下的选择偏差处理。在实践上，PSF 框架作为可重用基础设施，为后续评估研究提供了一套规范化、可控的实验平台，有助于推动社区围绕评估外部效度形成共识性的比较基准。在应用价值上，该方法为工业界 GenAI 系统的评估提供了一条切实可行的路径：无需大规模招募代表性人类标注者，仅需利用现有的有偏标注数据结合 LLM Persona 模拟，即可在统计意义上得到外部有效的系统质量估计，大幅降低了合规性评估的成本门槛。总体而言，本文的工作为"LLM-as-a-Judge"范式在实际落地中面临的分布偏移问题提供了坚实的理论基础与有效的工程解决方案，具有重要的学术价值与现实意义。
