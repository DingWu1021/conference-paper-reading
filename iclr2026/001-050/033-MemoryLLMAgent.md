# Exploratory Memory-Augmented LLM Agent via Hybrid On- and Off-Policy Optimization

> **Venue**: ICLR 2026 | **ArXiv**: 2602.23008 | **MemoryLLMAgent**

近年来，基于强化学习（RL）训练大语言模型（LLM）智能体已成为提升其在复杂环境中决策能力的主流范式。然而，探索问题始终是制约LLM智能体性能的核心瓶颈。现有方法（如GRPO、PPO等）主要依赖预训练知识中隐含的先验来指导探索，这在模型已有相关知识储备的任务上表现尚可，但一旦遇到需要发现全新状态、执行超出预训练分布范围动作序列的环境，这类方法便会出现严重的探索不足问题。以ScienceWorld为例，该基准测试包含19种科学实验任务，部分任务要求智能体按照精确的步骤组合完成实验，单纯依赖LLM内化的知识往往无法覆盖这些复杂的操作路径。在这种背景下，如何赋予LLM智能体更强的主动探索能力，使其能够在有限的交互次数内发现高回报的新状态，成为一个亟待解决的重要问题。

本文提出了EMPO²（Exploratory Memory-Augmented On- and Off-Policy Optimization），一个通过记忆增强与混合策略优化来强化LLM智能体探索能力的强化学习框架。其核心思路是将非参数化的记忆机制与参数化的策略优化相结合：在探索阶段，智能体利用记忆来发现并记录高价值的状态与轨迹；在学习阶段，通过同策略（on-policy）与异策略（off-policy）的混合更新，使策略参数既能在记忆辅助下稳定提升，又能在无记忆条件下保持鲁棒性。

EMPO²的记忆机制设计如下：每当一轮任务轨迹结束后，策略对该轨迹进行反思（reflection），生成自然语言形式的"经验提示"（reflective tips），并将其存入记忆缓冲区。在后续的推理步骤中，系统通过余弦相似度检索出与当前状态最相关的前10条提示，将这些提示附加到提示词（prompt）中，从而形成"记忆增强提示"（memory-augmented prompting）。正式地，标准推理时动作采样为 $a_{t+1} \sim \pi_\theta(\cdot \mid s_t, u)$，而记忆增强推理时采样为 $a_{t+1} \sim \pi_\theta(\cdot \mid s_t, u, \text{tips}_t)$，其中 $u$ 为任务指令，$\text{tips}_t$ 为检索到的历史经验提示集合。这一设计使智能体能够以非参数化的方式快速积累并复用探索经验，无需对模型权重进行修改即可在推理时利用过去的轨迹知识。

在策略更新方面，EMPO²提出了两种互补的优化模式。**同策略更新**（on-policy update）在记忆增强条件下同时完成轨迹采样与策略更新，重要性采样比率为 $\rho_\theta = \pi_\theta(a_t \mid s_t, u, \text{tips}_t) / \pi_{\theta_{\text{old}}}(a_t \mid s_t, u, \text{tips}_t)$，这保证了策略在有记忆辅助时能够稳健地学习。**异策略更新**（off-policy update）则采用"奖励引导的知识蒸馏"思路：轨迹在记忆增强提示条件下采样，但在策略更新时将提示移除，重新计算无提示条件下各动作的对数概率 $\ell_t^{\text{no-tips}} = \log \pi_\theta(a_t \mid s_t, u)$，以此损失更新模型参数。这一做法的意图是让基础策略（无记忆版本）通过模仿记忆增强策略所生成的高质量轨迹来内化记忆所带来的知识增益，相当于对有益行为进行选择性蒸馏（selective distillation）。

为了防止异策略更新时低概率动作带来的训练不稳定性，EMPO²引入了一种基于概率阈值 $\delta$ 的掩码机制（token masking）。具体而言，当策略对某一动作的概率低于 $\delta$ 时，对应的优势项（advantage term）被屏蔽，修正后的损失函数为：

$$\mathcal{L}_{\text{masked}} = \min\!\left(\rho_\theta A(a_t),\ \text{clip}(\rho_\theta, 1-\varepsilon, 1+\varepsilon)\, A(a_t)\right) \cdot \mathbf{1}\!\left[\pi_\theta(a_t \mid s_t, u) \geq \delta\right]$$

此外，系统还引入了基于新颖性的内在奖励（intrinsic reward）以鼓励状态空间探索：当智能体访问一个历史上较少出现的状态时，获得额外奖励 $r_{\text{intrinsic}} = 1/n$，其中 $n$ 为历史上相似状态的出现次数。这种内在激励与记忆检索形成协同：新颖状态触发更高的内在奖励，促使策略主动探索；探索所得经验被写入记忆，供后续轮次参考。框架整体在verl的GRPO实现基础上进行多步扩展，同策略与异策略更新概率分别为超参数 $p$（记忆推理概率）和 $q$（异策略更新概率），通过消融实验确定最优配置。

在实验评估方面，本文在ScienceWorld和WebShop两个具有代表性的序列决策基准上对EMPO²进行了系统测试。ScienceWorld包含19种科学实验任务，要求智能体在文本虚拟环境中执行精确的多步骤操作，对探索能力要求极高。EMPO²在该基准上取得了平均分75.9的成绩，而GRPO基线仅为33.2，提升幅度高达128.6%。相比之下，非参数化方法Reflexion仅得17.1分，离线强化学习方法Retrospex得33.8分，均远低于EMPO²。尤其值得注意的是，在初始阶段奖励为负的7个任务上，EMPO²最终全部达到了满分（100分），充分体现了其在困难探索任务上的优越性。在WebShop购物决策基准上，EMPO²得分为88.3±2.6，成功率为76.9%，分别高于GiGPO的86.2±2.6和75.2%，相对GRPO基线提升11.3%。

在分布外泛化（out-of-distribution generalization）实验中，EMPO²展现出令人印象深刻的迁移能力：在训练任务上完成参数更新后，仅通过记忆更新（无需任何梯度更新）在3个全新任务上进行适应，10步内平均提升136%。这表明EMPO²的记忆机制不仅在训练时提供探索引导，在推理部署阶段同样能够快速适应新环境，具有实际的工程价值。消融实验进一步确认了各组件的必要性：单独移除同策略更新或异策略更新均导致性能显著下降，两者相辅相成——同策略更新保证训练稳定性，异策略更新使基础策略内化记忆指导下的推理能力。

总体而言，EMPO²从探索瓶颈这一LLM智能体强化学习训练的根本难题出发，通过将轻量级非参数记忆机制与混合策略梯度优化有机结合，在不大幅增加模型参数或推理开销的前提下，显著提升了LLM智能体的探索广度与最终任务性能。其核心创新——异策略知识蒸馏——为解决"记忆辅助训练但无记忆部署"的不一致问题提供了一种优雅的解决方案，具有较强的普适性，可推广至其他需要探索的序列决策任务场景。这项工作对于构建能够在开放世界、长程任务中自主探索并持续学习的LLM智能体具有重要的参考意义。
