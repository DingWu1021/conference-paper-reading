# EgoTwin: Dreaming Body and View in First Person

> **Venue**: ICLR 2026 | **ArXiv**: 2508.13013 | **EgoTwin**

近年来，以第三人称（exocentric）视角为基础的视频生成技术取得了长足进步，从文本驱动的视频合成到可控相机轨迹生成，研究者们已经能够生成高质量、时序连贯的外部视角视频内容。然而，第一人称（egocentric）视角的视频生成却长期处于相对空白的状态。以头戴设备为代表的第一人称摄像场景具有其独特的物理约束：摄像机刚性地固定在佩戴者头部，因此摄像机的运动轨迹与佩戴者的头部姿态之间存在着直接的、强耦合的对应关系。这种耦合关系使得第一人称视频生成不能简单地套用外部视角视频生成的框架，而必须同时建模视频内容与人体运动两个模态。EgoTwin 正是在这一背景下被提出的，它开创性地定义了"联合第一人称视频与人体动作生成"这一新任务，旨在从文本描述出发，同步生成与佩戴者身体运动高度一致的第一人称视频序列和全身人体动作。

这项工作的研究动机来源于两个核心挑战。第一个挑战称为**视角对齐**（Viewpoint Alignment）：由于摄像机固定于头部，生成视频的相机轨迹必须与所生成人体运动中的头部轨迹精确吻合，即视频中的视点变化应当完全由头部的旋转和平移决定。第二个挑战称为**因果交互**（Causal Interplay）：在真实的第一人称场景中，观察者的视觉感知（当前帧内容）会影响其随后的动作决策，而动作的执行又会改变下一帧的视觉内容，两者之间形成一个类似控制论中"观察-行动"循环的递归因果关系。忽视这一因果关系，就无法保证生成视频与生成动作在时序层面的细粒度同步，导致两者出现明显的不一致，例如视频中的手部位置与动作序列中的手部关节坐标发生偏移。

为应对上述挑战，EgoTwin 提出了一套基于多模态扩散变换器（MM-DiT）架构的联合生成框架，并引入了两项关键技术创新。

第一项创新是**以头部为中心的动作表征**（Head-Centric Motion Representation）。传统的人体动作表示方法通常以髋部（根关节）为参考坐标系，记录各关节相对于根关节的位置与旋转。然而，这种以根关节为中心的表示方式隐藏了头部姿态信息——要从中恢复摄像机的视角，需要经过多步骤的运动学链式计算，这对神经网络而言极难精确建模。EgoTwin 因此提出将动作表示锚定于头部关节，显式地包含头部的绝对旋转（6D表示）、相对旋转（6D表示）、绝对位置（3D向量）、相对位置（3D向量），以及其余各关节在头部坐标系下的位置、速度和旋转信息。这种表示方式将第一人称视角所需的核心信息——头部姿态——直接暴露给视频生成分支，从根本上简化了两模态之间的对齐问题。

第二项创新是**控制论启发的交互机制**（Cybernetics-Inspired Interaction Mechanism）。在 MM-DiT 的注意力操作中，EgoTwin 对视频 token 与动作 token 之间的交叉注意力施加了结构化的因果掩码：视频 token 只关注时间上在其之前的动作 token，建模"过去的动作决定当前的观察"这一前向动力学；动作 token 则关注时间上当前及未来的视频 token，建模"当前及未来的观察变化推断当前应采取的动作"这一逆向动力学。这种不对称的双向注意力设计，将控制论中的观察-行动反馈回路显式地编码进了模型的注意力计算，使视频生成与动作生成之间的时序依赖关系得到了精确建模。

除上述两项核心创新外，EgoTwin 还引入了**异步扩散**（Asynchronous Diffusion）训练策略：对视频分支和动作分支分别独立采样扩散时间步 $t_v$ 和 $t_m$，而非强制两者同步。这一设计允许两个模态在去噪过程中以各自合适的速率演化，更好地捕捉它们在时序上复杂而多样的依赖关系。

在架构层面，EgoTwin 由三个分支组成：以 T5-XXL 为基础的文本编码分支（在联合训练阶段冻结权重）、由 42 层处理 3D VAE 编码视频的视频生成分支，以及由 21 层（对应视频分支的下半部分）处理动作序列的动作生成分支。动作分支的层数和维度相对较少，反映了动作模态相比视频模态结构上的简洁性。动作 VAE 的训练损失对头部三维位置、头部六维旋转、关节三维位置、关节六维旋转四个分量分别施加权重，以避免因维度数量不同而导致的梯度失衡问题。整个训练分三阶段进行：首先训练动作 VAE；其次在冻结文本分支的前提下进行文本条件动作生成的预训练；最后进行文本-视频-动作的联合训练，并以 10% 的概率丢弃文本嵌入以支持无条件生成建模。

在数据集方面，EgoTwin 使用了 Nymeria 数据集——一个大规模真实场景数据集，涵盖室内外多样日常活动场景，包含由 Project Aria 头戴设备采集的第一人称视频、Xsens 动作捕捉系统记录的全身动作，以及对应的文字描述，共约 17 万个同步的文本-视频-动作三元组样本。

在评估指标方面，除了常规的视频质量指标（图像 FID 即 I-FID、视频 FID 即 FVD、文本-视频语义相似度 CLIP-SIM）和动作质量指标（动作 FID 即 M-FID、检索精度）之外，EgoTwin 还专门设计了三个针对视频-动作一致性的评估指标：**平移误差**（Translation Error，衡量相机位姿中心与头部位置之间的平均距离）、**旋转误差**（Rotation Error，衡量相机朝向与头部朝向之间的角度差异）、以及**手部 F 值**（Hand F-Score，评估视频与动作中手部可见性的一致程度）。

在定量实验结果上，EgoTwin 相较于基线方法 VidMLD 取得了全面显著的提升。在视频质量方面，I-FID 从 157.86 降至 98.17（提升约 38%），FVD 从 1547.28 降至 1033.52（提升约 33%）。在视频-动作一致性方面，平移误差从 1.28 降至 0.67（提升约 48%），旋转误差从 1.53 降至 0.46（提升约 70%），手部 F 值从 0.36 提升至 0.81（提升约 125%）。消融实验进一步验证了各设计组件的必要性：去除头部中心动作表征后，平移误差上升 43%（从 0.67 至 0.96）；去除因果交互机制后，旋转误差上升 93%（从 0.46 至 0.89）；去除异步扩散后，各指标均有不同程度的下降。

值得关注的是，EgoTwin 所学习到的视频与动作联合分布，还支持多种下游条件生成任务：给定文本和视频可生成对应动作，给定文本和动作可生成对应视频。更进一步，由于动作模态中包含了头部的绝对位姿信息，可以直接从生成的动作序列中提取相机位姿，进而与三维高斯点云散射（3D Gaussian Splatting）方法相结合，实现具有真实人体-场景交互的三维场景重建，展示了该框架在具身智能和增强现实场景中的广阔应用潜力。

综合来看，EgoTwin 的研究意义体现在三个层面。其一，它在学术上定义并系统解决了联合第一人称视频与人体动作生成这一此前尚未被充分探索的新任务，填补了第一人称视觉生成领域的空白。其二，其提出的以头部为中心的动作表征和控制论启发的注意力掩码机制，是具有一般性的方法论贡献，对其他需要精细时序同步和因果依赖建模的多模态生成任务具有直接的借鉴价值。其三，从应用角度看，高质量的联合第一人称视频-动作生成对于增强现实、具身智能体训练、人体行为理解等前沿领域均具有重要的实用价值，为相关研究提供了强有力的数据合成工具与基础模型。
