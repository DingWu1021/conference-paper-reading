# Mean Flow Policy with Instantaneous Velocity Constraint for One-step Action Generation

> **Venue**: ICLR 2026 | **ArXiv**: 2602.13810 | **MeanFlowPolicy**

基于流匹配（flow matching）的生成式策略模型在机器人操控领域取得了显著进展，其核心思路是将动作生成建模为从噪声分布到目标动作分布的概率流变换过程。然而，现有的流策略方法普遍面临一个难以回避的矛盾：若要保证策略的表达能力，就必须通过多步积分来采样动作，这在训练阶段和推理阶段都带来了大量计算开销；而若将采样步数压缩到一步，策略的表达能力又会大幅下降，难以覆盖复杂多模态的动作分布。这一效率与表达能力之间的本质张力，严重制约了流策略在实时机器人控制场景中的实用化进程。本文作者来自清华大学和加州大学伯克利分校（BAIR），针对上述挑战提出了一种名为均值速度策略（Mean Velocity Policy，MVP）的新型生成式策略框架，旨在从理论和实践两个层面同时解决效率与表达能力的权衡难题。该工作被ICLR 2026接收为口头报告（Oral），充分体现了其在学术界的重要价值。

在方法论层面，MVP的关键创新在于将策略函数从建模瞬时速度场（instantaneous velocity field）转变为建模均值速度场（mean velocity field）。传统流策略通过对常微分方程进行数值积分来生成动作：$a(1) = a(0) + \int_0^1 v(a(\tau), \tau, s)\, d\tau$，其中 $a(0) \sim \mathcal{N}(0, I)$ 为初始噪声，$v$ 为瞬时速度场，$s$ 为当前状态观测。这一积分过程天然需要多步离散化计算。而MVP改为直接参数化时间区间 $[t, r]$ 上的均值速度：

$$u(a(t), t, r, s) \triangleq \frac{1}{r-t} \int_t^r v(a(\tau), \tau, s)\, d\tau$$

在推理时，动作生成仅需一步完成：$a(1) = a(0) + u^*(a(0), 0, 1, s)$，从而将多步积分压缩为单步前向传播，同时保留了流模型对复杂分布的建模能力。为了训练均值速度场，作者推导了均值流恒等式（mean flow identity）。对均值速度定义关于时间 $t$ 求导，可得：

$$-u(a(t), t, r, s) + (r-t)\frac{d}{dt}u(a(t), t, r, s) = -v(a(t), t, s)$$

基于此恒等式，训练目标（均值流损失）定义为：

$$\mathcal{L}_{\text{MF}}(\theta) = \mathbb{E}_{t, r<t, a(t)} \left\| u_\theta(a(t), t, r, s) - \text{sg}\!\left(v - (t-r)\frac{d}{dt}u_\theta(a(t), t, r, s)\right) \right\|_2^2$$

其中 $\text{sg}(\cdot)$ 表示停止梯度操作符。然而，作者通过理论分析（定理2）指出，均值流恒等式作为一阶常微分方程，本身缺乏显式边界条件，因此其解并不唯一：满足该方程的任意解都存在形如 $\Delta_u(a(t), t, r) = C(a, r)/(r-t)$ 的累积误差偏置，其中 $C(a,r)$ 是与时间 $t$ 无关的积分常数。这一理论发现揭示了纯均值流训练的内在缺陷——即使训练损失收敛，模型仍可能因积分常数的存在而产生系统性偏差，严重影响单步生成的精确性。

为解决上述边界条件缺失问题，作者提出了瞬时速度约束（Instantaneous Velocity Constraint，IVC）。IVC的核心思想是：当区间长度 $r-t \to 0$ 时，均值速度应退化为瞬时速度，即 $u(a(t), t, t, s) = v(a(t), t, s)$。因此，额外引入如下约束损失：

$$\mathcal{L}_{\text{IVC}}(\theta) = \mathbb{E}_{t, a(t)} \left\| u_\theta(a(t), t, t, s) - v \right\|_2^2$$

作者在定理3中严格证明，联合最小化 $\mathcal{L}_{\text{IVC}}$ 能够迫使积分常数 $C(a,r)$ 趋于零，从而消除累积误差，保证均值速度场解的唯一性与正确性。最终的完整训练目标为：

$$\mathcal{L}_{\text{policy}}(\theta) = \mathcal{L}_{\text{MF}}(\theta) + \lambda \mathcal{L}_{\text{IVC}}(\theta)$$

其中 $\lambda$（默认值为1.0）为IVC损失的权重系数。在强化学习框架下，策略改进通过"生成多个候选动作 + Q值选择最优动作"的范式实现，作者同样给出了MVP策略改进保证（定理1），明确了采样多样性对最终策略价值的正向贡献以及学习误差对策略提升的负向影响，从理论上支撑了IVC设计的必要性。

在实验设计方面，作者在两个标准机器人操控基准上系统评估了MVP：Robomimic基准包含三个任务（Lift、Can、Square），每个任务提供300条多人类示范轨迹；OGBench基准包含六个积木摆放任务（double-task和triple-task各三个难度级别），以错放积木数量为稀疏奖励信号。主要对比基线包括：流Q学习方法FQL（多步流策略及其单步蒸馏变体）、最佳候选流方法BFN（10步流+候选动作选择）以及动作分块扩展方法QC。实验结果表明，MVP在整体平均成功率上达到88%，显著优于FQL（44%）、BFN（52%）和QC（86%）。在最具挑战性的长期任务上，MVP的优势尤为突出：Robomimic-Square任务成功率93%（对比FQL的12%）；OGBench Cube-triple-task4任务成功率52%（对比FQL的0%、BFN的2%）。计算效率方面，MVP的训练速度达到每秒153.6次迭代，比最慢基线BFN快约126%，比FQL快约41%；推理延迟仅约10.93毫秒（CPU，无JIT优化），与单步基线FQL相当，远低于需要多步采样的QC（113.22毫秒）和BFN（117.3毫秒）。消融实验进一步验证了IVC的关键作用：在Cube-triple-task4任务上，移除IVC（$\lambda=0$）后成功率从0.52下降至0.30，与理论预测高度吻合，证明边界条件约束对提升模型精确性不可或缺。此外，将现有基线方法直接压缩为单步推理后，其在高难度任务上的成功率接近于零，而MVP凭借均值速度场的固有表达能力，仍能取得0.71和0.52的成功率，充分说明方法设计的有效性。

综合来看，本文提出的MVP框架在生成式强化学习策略领域做出了重要贡献。其核心价值在于：从理论层面揭示了均值流训练缺乏边界条件导致解不唯一的根本缺陷，并通过优雅的IVC设计加以修复；在算法层面实现了单步动作生成与高表达能力的有机统一；在工程层面大幅提升了训练与推理速度，为实时机器人控制的落地应用提供了有力支撑。作者也坦诚指出，由于训练过程中需要计算Jacobian-向量积（JVP）来实现均值速度对时间的导数，GPU显存消耗有所增加，这是当前方法的主要局限。未来工作计划在真实机器人平台和更广泛任务域上进行验证，进一步拓展MVP的应用边界。该工作为基于流匹配的生成式策略研究指明了一个清晰的理论方向，即如何通过恰当的边界条件设计来保证快速采样与分布表达能力的兼得，对后续相关研究具有重要的启发意义。
